hdfsGetDefaultBlockSizeAtPath() Supported by libMapRClient for hadoop-2.x Gets the block size of a file at the specified path. Signature tOffset hdfsGetDefaultBlockSizeAtPath(hdfsFS fs, const char *path) Parameters Parameter Description fs The handle of the filesystem. Obtain this handle with one of the hdfsConnect() APIs. path The location and name of the file. Return Value Returns 256 MB. Check errno for error codes and meanings.
