Common Options Describes options that are available for many commands. The following options are available for most commands in the command-line context. Option Description -noheader When displaying tabular output from a command, omits the header row. -long Shows the entire value. This is useful when the command response contains complex information. When -long is omitted, complex information is displayed as an ellipsis (...). -json Displays command output in JSON format. When -json is omitted, the command output is displayed in tabular format. -cli.loglevel Specifies a log level for API output. Legal values for this option are: * DEBUG * INFO * ERROR * WARN * TRACE * FATAL
