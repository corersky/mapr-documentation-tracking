license add Adds a license. Permissions required: fc or a. The license can be specified either by passing the license string itself to license add, or by specifying a file containing the license string. Syntax CLI maprcli license add [ -cluster ] [ -is_file true|false ] -license REST Request Type POST Request URL http[s]://:/rest/license/add? Parameters Parameter Description cluster The cluster on which to run the command. is_file Specifies whether the license specifies a file. If false, the license parameter contains a long license string. license The license to add to the cluster. If -is_file is true, license specifies the filename of a license file. Otherwise, license contains the license string itself. Examples Note: After obtaining a valid license file from your MapR sale representative, copy the license file to a cluster node, for example to /tmp/license.txt. To add a license from a file: CLI maprcli license add -is_file true -license /tmp/license.txt REST https://abc.sj.us:8443/rest/license/add?is_file=true&license=%2F tmp%2Flicense.txt
