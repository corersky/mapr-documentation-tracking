dialhome enable Enables Dial Home on the cluster. Permissions required: fc or a Syntax CLI maprcli dialhome enable -enable 0|1 REST http[s]://:/rest/dialhome/enable Parameters Parameter Description enable Specifies whether to enable or disable Dial Home: * 0 - Disable * 1 - Enable Output A success or failure message. Sample output # maprcli dialhome enable -enable 1 enabled 1 # maprcli dialhome status enabled 1 Examples Enable Dial Home: CLI maprcli dialhome enable -enable 1 REST https://r1n1.sj.us:8443/rest/dialhome/enable?enable=1
