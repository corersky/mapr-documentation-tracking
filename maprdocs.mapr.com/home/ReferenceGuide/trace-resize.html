trace resize Resizes the trace buffer. Syntax CLI maprcli trace resize [ -host ] [ -port ] -size REST None. Parameters Parameter Description host The IP address of the node on which to resize the trace buffer. Default: localhost port The port to use. Default: 5660 size The size of the trace buffer, in kilobytes. Default: 2097152 Minimum: 1 Examples Resize the trace buffer to 1000 CLI maprcli trace resize -size 1000
