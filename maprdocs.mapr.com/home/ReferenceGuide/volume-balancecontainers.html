volume balancecontainers Balance the containers or stop the balancing of containers associated with the volume. Syntax CLI maprcli volume balancecontainers [ -cluster cluster_name ] -name [ -cancel true ] REST API N/A Parameters Parameter Description cancel Stop the balancing of containers associated with the volume. Value must be true to cancel balancing activities. cluster The name of the cluster to run the command on. name The name of the volume. Examples Start balancing the containers associated with a volume: maprcli volume balancecontainers -name sampleVol Stop balancing the containers associated with a volume: maprcli volume balancecontainers -name sampleVol -cancel true
