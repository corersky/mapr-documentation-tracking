mrconfig info orphanagecount The mrconfig info orphanagecount command displays orphan entries for a given container (specified by ID). Syntax /opt/mapr/server/mrconfig info orphanagecount Parameters Parameter Description cid The ID of the container. Examples Display the number of orphan entries for container 2067: ~# /opt/mapr/server/mrconfig info orphanagecount 2067 ---------------------- Time: 2017-03-31 18:00:49,1085 Instance 5660 ---------------------- orphanagecount cid 2067 count 812
