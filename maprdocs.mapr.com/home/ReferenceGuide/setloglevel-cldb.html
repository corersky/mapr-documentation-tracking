setloglevel cldb Sets the log level on the CLDB service. Permissions required: fc or a. Syntax CLI maprcli setloglevel cldb -classname -loglevel DEBUG|ERROR|FATAL|INFO|TRACE|WARN -node -port REST http[s]://:/rest/setloglevel/cldb? Parameters Parameter Description classname The name of the class for which to set the log level. The class can be at the package level or a specific class. Contact MapR Support for this parameter. loglevel The log level to set. Default: INFO * DEBUG * ERROR * FATAL * INFO * TRACE * WARN node The node on which to set the log level. port The CLDB port. Default: 7222 Examples CLI maprcli setloglevel cldb -classname com.mapr.fs.cldb.CLDBServer -loglevel debug -node centos26.lab REST https://centos26.lab:8443/rest/setloglevel/cldb?classname=com.mapr.fs.cldb.CLDBS erver&loglevel=debug&node=centos26.lab
