volume recall Recalls the offloaded data for the specified volume. Permissions Required The user running the command must have one of the following: * Full control (fc) on the cluster or volume * Volume edit permissions Syntax CLI maprcli volume recall [ -cluster cluster_name ] -name REST Request Type POST Request URL http[s]://:/rest/volume/recall? Parameters Parameter Description cluster The name of the cluster on which to run the command. name The name of the volume.
