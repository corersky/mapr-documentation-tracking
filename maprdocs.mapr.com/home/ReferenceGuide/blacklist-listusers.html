blacklist listusers Lists blacklisted users on a specific cluster. By default, this command lists users that have been blacklisted from the cluster where the command is run. There is no REST equivalent command. For information about blacklisting, see [1]How Tickets Work. Syntax CLI maprcli blacklist listusers [ -cluster ] REST N/A Parameters Parameter Description cluster Name of the cluster to blacklist the user on. Examples Show blacklisted users for the cluster my.cluster.com: CLI maprcli blacklist listusers -cluster my.cluster.com References 1. file://localhost/root/docsync/tmp/maprdocs.mapr.com/home/SecurityGuide/HowTicketsWork.html#TicketsandCertificates-Ho_26281159-d3e369
