yarn debugcontrol used to save additional DEBUG logs for scheduling in YARN to a separate file without restarting the RM. The logs are saved in /opt/mapr/Hadoop/Hadoop/logs/yarn-mapr-scheduling-debug.log Syntax yarn debugcontrol [-addapp ] | [-addqueue ]| [-removeapp ] | [-removequeue ] | [-getapps ] | [-getqueues ] Parameters Parameter Description -addapp enables additional scheduling DEBUG on the application -addqueue enables additional scheduling DEBUG on the queue -removeapp disable addition scheduling DEBUG on the application -removequeue disable addition scheduling DEBUG on the queue -getapps lists the applications with additional scheduling DEBUG -getqueues lists the queues with additional scheduling DEBUG
