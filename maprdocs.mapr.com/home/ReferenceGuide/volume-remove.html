volume remove Removes the specified volume or mirror. Permissions required: d or fc on the volume. Syntax CLI maprcli volume remove [ -cluster ] [ -force 0|1 ] -name [ -filter ] REST Request Type POST Request URL http[s]://:/rest/volume/remove? Parameters Parameter Description cluster The cluster on which to run the command. force Forces the removal of the volume, even if it would otherwise be prevented. name The volume name. filter All volumes with names that match the filter are removed. Examples CLI maprcli volume remove -name testVolume REST https://abc.sj.us:8443/rest/volume/remove?name=testVolume
