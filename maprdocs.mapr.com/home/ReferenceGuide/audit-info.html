audit info Displays whether auditing of cluster-management operations and auditing data-access operations are enabled. Also, displays the maxSize and retention values for these two levels of auditing. Only the mapr user for the cluster can run this command. For more information about the mapr user, see [1]Managing Users and Groups. Syntax CLI maprcli audit info [ -cluster ] -json REST http[s]://:/rest/audit/info?json Parameter Parameter Description cluster The name of the cluster on which to run the command. json Displays command output in JSON format. When -json is omitted, there is no output. Example Output This output shows that auditing of operations on data and auditing of cluster-level operations are both enabled.
