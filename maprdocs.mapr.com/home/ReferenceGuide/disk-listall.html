disk listall Lists all disks. Syntax CLI maprcli disk listall [ -cluster ] [ -limit ] [ -output terse|verbose ] [ -start ] REST http[s]://:/rest/disk/listall? Parameters Parameter Description cluster The cluster on which to run the command. limit The number of rows to return, beginning at start. Default: 0 output Always the string terse. start The offset from the starting row according to sort. Default: 0 Output Information about all disks. See the [1]Disk Fields table.
