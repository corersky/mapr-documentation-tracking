cluster feature list Allows features to be listed. Used after upgrading. Syntax CLI maprcli cluster feature list [ -name ] [ -enabled ] [ -disabled ] REST N/A Parameters Parameter Description name Lists the named feature. Displays whether the feature is enabled or disabled along with the feature name. enabled Lists only the enabled features. No value is taken. disabled Lists only the disabled features. No value is taken. Examples Lists the disabled features. Example command: maprcli cluster feature -disabled Example output: enabled name false mfs.feature.volume.upgrade Lists all the features.
