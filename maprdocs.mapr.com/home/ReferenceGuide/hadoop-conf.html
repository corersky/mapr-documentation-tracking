hadoop conf The hadoop conf command outputs the configuration information for this node to standard output. Syntax hadoop [ generic options ] conf Examples Displaying the configured value of a specific parameter [user@hostname ~]$ hadoop conf | grep mapreduce.map.memory.mb mapreduce.map.memory.mb1024mapred- site.xml The above command returns 1024 as the configured value of the mapreduce.map.memory.mb parameter. Dumping a node's configuration to a text file [user@hostname ~]$ hadoop conf | grep ... >nodeconfiguration.txt The above command creates a text file named nodeconfiguration.txt that contains the node's configuration information.
