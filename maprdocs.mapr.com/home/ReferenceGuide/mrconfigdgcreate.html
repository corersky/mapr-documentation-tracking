mrconfig dg create The mrconfig dg create commands let you create disk groups (after you initialize disks with the [1]mrconfig disk init command and add them to the node with the [2]mrconfig disk load command). You can create a disk group with one of two formats: * Use the [3]mrconfig dg create raid0 command to create a striped disk group with a [4]RAID 0 format. * Use the [5]mrconfig dg create concat command to create a [6]concatenated disk group (one disk after another). After you create disk groups you will be ready to [7]create storage pools on the disk groups. See [8]mrconfig for instructions about running mrconfig commands.
