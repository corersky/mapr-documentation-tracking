dump supportdump Collects logs and other information about the node to help troubleshoot issues. Syntax CLI maprcli dump supportdump [ -cluster ] [ -nodes ] [ - params ] [ -zkconnect ] REST N/A Parameters Parameter Description cluster Cluster name. nodes Node names for which support dump is needed. Space separated. Default: all params Parameter string to create a dump. zkconnect ZK connection string.
