volume upgradeformat Upgrades and old-type volume to a new-type volume, which can in turn be used as a promotable mirror volume. Permissions required: m or fc on the volume. Syntax CLI maprcli volume upgradeformat [ -cluster ] -name REST http[s]://:/rest/volume/upgradeformat? Parameters Parameter Description cluster The cluster on which to run the command. name The volume name. Examples CLI maprcli volume upgradeformat -name vol999 -json REST https://10.10.82.23:8443/rest/volume/upgradeformat?name=vol999
