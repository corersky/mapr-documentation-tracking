stream delete Deletes the specified stream. Deleted streams cannot be recovered unless they were previously replicated. Producers are no longer able to publish messages to topics in the stream, and consumers are no longer able to read messages from topics in the stream. Permissions Required To run this command, your user ID must have the following permissions: * [1]readAce and [2]writeAce on the volume * [3]lookupdir on directories in the path Note: The mapr user is not treated as a superuser. MapR-ES does not allow the mapr user to run this command unless that user is given the relevant permission or permissions with access-control expressions. Syntax CLI maprcli stream delete -path REST http[s]://:/rest/stream/delete?path= Parameters Parameter Description path The path and name of the stream to delete.
