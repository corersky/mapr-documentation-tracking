entity info Displays information about an entity. Syntax CLI maprcli entity info [ -cluster ] -name [ -output terse|verbose ] -type REST Request Type GET Request URL http[s]://:/rest/entity/info? Parameters Parameter Description cluster The cluster on which to run the command. name The entity name. Obtain the entity name by running the maprcli entity list command. output Whether to display terse or verbose output. type The entity type. Obtain the entity type by running the maprcli entity list command.
