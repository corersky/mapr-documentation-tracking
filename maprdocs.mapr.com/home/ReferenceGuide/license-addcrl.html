license addcrl Adds a certificate revocation list (CRL). Permissions required: fc or a. Syntax CLI maprcli license addcrl [ -cluster ] -crl [ -is_file true|false ] REST http[s]://:/rest/license/addcrl? Parameters Parameter Description cluster The cluster on which to run the command. crl The CRL to add to the cluster. If file is set, crl specifies the filename of a CRL file. Otherwise, crl contains the CRL string itself. is_file Specifies whether the license is contained in a file. Examples CLI maprcli license addcrl -crl crl.txt -is_file true REST https://centos26.lab:8443/rest/license/addcrl?crl=crl.txt&is_file=true
