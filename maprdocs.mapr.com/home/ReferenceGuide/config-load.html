config load Displays information about the cluster configuration. Syntax CLI maprcli config load [ -cluster ] [ -keys ] REST http[s]://:/rest/config/load? Parameters Parameter Description cluster The cluster for which to display values. keys This parameter is used to specify which information to display. Comma-separated fields are used to display values; see the [1]Configuration Fields table. Output Information about the cluster configuration. See the [2]Configuration Fields table.
