node cldbmaster Returns the address of the master CLDB node. The node cldbmaster API returns the server ID and hostname of the node serving as the CLDB master node. Syntax CLI maprcli node cldbmaster [ -cluster ] REST Request Type GET Request URL http[s]://:/rest/node/cldbmaster[?] Parameters Parameter Description cluster The name of the cluster for which to return the CLDB master node information. Examples Return the CLDB master node information for the cluster my.cluster.com: CLI maprcli node cldbmaster -cluster my.cluster.com REST https://abc.sj.us:8443/rest/node/cldbmaster?cluster=my.cluster.com
