license listcrl Lists certificate revocation lists (CRLs) on the cluster. Permissions required: login. Syntax CLI maprcli license listcrl [ -cluster ] REST http[s]://:/rest/license/listcrl? Parameters Parameter Description cluster The cluster on which to run the command. Examples CLI maprcli license listcrl -cluster my.test.cluster REST https://centos26.lab:8443/rest/license/listcrl?cluster=my.test.cluster
