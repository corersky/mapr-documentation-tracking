license listcrl Lists certificate revocation lists (CRLs) on the cluster. Permissions required: login. Syntax CLI maprcli license listcrl [ -cluster ] REST Request Type GET Request URL http[s]://:/rest/license/listcrl[?] Parameters Parameter Description cluster The cluster on which to run the command. Examples CLI maprcli license listcrl -cluster my.test.cluster REST https://abc.sj.us:8443/rest/license/listcrl?cluster=my.test.cluster
