Switching from Zero Configuration to Manual or Automatic Failover You can change your ResourceManager failover implementation from zero configuration to manual or automatic failover by re-configuring all the cluster and client nodes. For more information, see [1]Configuring Manual Failover for the Resource Manager or [2]Configuring Automatic Failover for the Resource Manager.
