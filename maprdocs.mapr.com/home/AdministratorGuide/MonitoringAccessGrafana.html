Access the Grafana UI You can launch the Grafana UI from the MCS or directly from a web browser. 1. Use one of the following methods to launch the Grafana UI: + In MCS, go to the Services page and click Grafana to launch the Grafana UI in another tab. + From a web browser, launch the following URL: http://:3000 2. Provide user credentials. See [1]Logging on to Grafana. 3. Click Log in.
