Configuring Settings You can configure the settings in the UI for scanning and uploading the files. 1. Log in to the UI and select Settings from the [gearMenu.png] dropdown menu. [dropdownMenu.png] The File Migration Settings page displays. 2. Optionally, set the following for file upload: + Full Scan Frequency + Upload Scan Frequency + Completion Scan Frequency + Minimum Wait Before Upload + Maximum Active Uploads + Minimum Wait After Failure + Maximum Retries Per File + Maximum System Retries + Minimum Idle Time Before Error See [1]FileMigrate.properties for more information. 3. Set the following to enable the service to connect and upload to AWS S3: + Proxy Host + Proxy Port + Proxy Username + Proxy Password See [2]FileMigrate.properties for more information. 4. Click Save Changes for the changes to take effect.
