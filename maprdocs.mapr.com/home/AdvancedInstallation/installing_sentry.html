Installing Sentry This topic includes instructions for using package managers to download and install Sentry from the MEP repository. Set up the MEP repository, if you have not done so. For instructions on setting up the MEP repository, see [1]Step 10: Install Ecosystem Components Manually. Complete the following steps as root or using sudo to install Sentry: 1. Set the variable JAVA_HOME. The mapr-sentry package works with JDK versions 1.7 and 1.8. In the following example, java-7-oracle is used: # export JAVA_HOME=/usr/lib/jvm/java-7-oracle 2. On the nodes designated to run Sentry, install mapr-sentry: + On Ubuntu: apt-get install mapr-sentry + On RedHat/CentOS: yum install mapr-sentry + On SUSE: zypper install mapr-sentry 3.
