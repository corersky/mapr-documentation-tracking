Checking the MapR Installer Version This page describes how to read the MapR Installer version. To check the version of the currently installed MapR Installer: 1. From the MapR Installer web-based interface, click Support. The drop-down menu shows the MapR Installer version: [installer_version.png] 2. Optional: Check your version against the recently released versions to see if a newer version is available. For a list of recently released versions, see [1]MapR Installer Updates. To update the MapR Installer, see [2]Updating the MapR Installer.
