Installing MapR-XD Describes how to install MapR-XD software with or without the MapR Installer. The steps for installing MapR-XD are the same as the steps for installing MapR software, with some exceptions noted below. For more information about MapR-XD, see [1]MapR-XD. Before Installing MapR-XD Using the MapR Installer Regardless of the method you use to install MapR-XD, before installing, you should review the information in these topics: * [2]MapR Repositories and Packages * [3]Preparing each Node Installing MapR-XD Using the MapR Installer Use these steps to install MapR-XD using the web-based MapR Installer: 1. Download the MapR Installer. See [4]MapR Installer. 2. On the Version & Services page of the MapR Installer, specify these values: + MapR Version: 6.0.0 or later. + Edition: Converged Enterprise Edition. + License Option: Apply the MapR-XD license after the installation completes. + MEP Version: The MEP version is pre-selected. + Auto-Provisioning Template: MapR-XD. You do not need to select any services. 3. Click Next to advance through the menus. 4. On the Monitoring page, enable monitoring for metrics and logs. 5. When the MapR Installer indicates that the installation is complete, go to [5]After Installing a MapR-XD Cluster. Installing MapR-XD without Using the MapR Installer Use the following information to install MapR-XD using manual steps. You can use the manual installation steps described in [6]Installing without the MapR Installer to install the MapR 6.0.0 or later packages. Note these considerations for some steps: Step 3 Install these MapR 6.0.0 or later packages at a minimum: * mapr-core * mapr-fileserver * mapr-cldb * mapr-nfs * mapr-webserver * mapr-zookeeper For information about packages and dependencies, see [7]Packages and Dependencies for MapR Software (MapR 6.0.1). Step 7 Apply the MapR-XD license. Step 8 Complete the steps to install Metrics Monitoring. Step 9 Complete the steps to install Log Monitoring. Step 10 Skip this step. You do not need to install any ecosystem components. Step 11 Complete this step. Running configure.sh with the -R option is required. See [8]After Installing a MapR-XD Cluster.
