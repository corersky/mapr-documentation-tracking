Upgrading the MapR-ES Python Client This section describes how to upgrade the MapR-ES Python Client without the MapR Installer. Execute the following command as root or using sudo: pip install -U --global-option=build_ext --global-option="--library-dirs=/opt/ma pr/lib" --global-option="--include-dirs=/opt/mapr/include/" http://package.mapr.com/releases/MEP/MEP-/mac/mapr-streams-python-.tar.gz Alternatively: pip install --upgrade mapr-streams-python
