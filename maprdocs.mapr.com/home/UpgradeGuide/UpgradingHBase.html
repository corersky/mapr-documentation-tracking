Upgrading HBase Client and Tools This section describes how to upgrade HBase Client and other tools without the MapR Installer. Service Description HBase Client Upgrading the HBase Client upgrades both the HBase Shell and the HBase APIs supported for use with MapR-DB binary tables and MapR-DB JSON tables. HBase Thrift Gateway HBase Thrift Gateway includes an API and a service that accepts Thrift requests to connect to MapR-DB tables. See [1]Upgrading HBase Thrift Gateway for upgrade information. HBase REST Gateway HBase REST Gateway includes an API and a service that accepts REST requests to connect to MapR-DB tables. See [2]Upgrading HBase REST Gateway for upgrade information. AsyncHBase Libraries AsyncHBase library provides asynchronous Java APIs to access MapR-DB tables. See [3]Upgrading AsyncHBase Libraries for upgrade information.
