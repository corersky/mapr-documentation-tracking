Upgrade the MapR Client on Windows This section provides instructions on how to upgrade the MapR Client on Windows. When you upgrade the MapR client on Windows, you need to rename the existing client directory, install the new version, and then merge the configuration files. 1. Rename the existing client installation directory. For example, you can rename \opt\mapr to \opt_old\mapr. 2. Complete the installation steps in [1]Installing the MapR Client. 3. To retain existing configurations and accepts new defaults, merge the contents of the following directories in the previous installation with the ones in the new installation: + %MAPR_HOME%\hadoop\hadoop-2.x.x\etc\hadoop + %MAPR_HOME%\hadoop\hadoop-0.20.0\conf References 1. file://localhost/root/docsync/tmp/maprdocs.mapr.com/home/AdvancedInstallation/SettingUptheClient-windows.html
