Upgrading Drill This section describes how to upgrade Drill without the MapR Installer. Complete the following step on Drill server and client nodes as root or using sudo to upgrade Drill without the MapR Installer: Issue the command appropriate for your system on each Drill node: * RedHat/CentOS yum update mapr-drill * Ubuntu sudo apt-get install mapr-drill * SUSE zypper update mapr-drill Note: SUSE is supported as of Drill 1.9.0-1703 and Drill 1.10.0-1703.
