Upgrading Pig This section describes how to upgrade Pig without the MapR Installer. Complete the following steps to upgrade Pig without the MapR Installer. Run the following command to upgrade Pig using a package manager: On RedHat and CentOS yum update mapr-pig On Ubuntu apt-get install mapr-pig On SUSE zypper update mapr-pig To apply custom configurations to the new version, migrate any custom configuration settings into the new default files in the conf directory.
