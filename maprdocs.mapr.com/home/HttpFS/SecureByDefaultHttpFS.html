Authentication on Secure Clusters for HttpFS In MEP 3.0, HttpFS features several authentication methods on secure clusters. HttpFS authentication, such as MapR native security (MapR-SASL), Kerberos (for which [1]additional configuration is required), or plain (using PAM) is determined automatically.
