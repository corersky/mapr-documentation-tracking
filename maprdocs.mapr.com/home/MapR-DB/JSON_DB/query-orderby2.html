Querying with Order By The examples in this section query a document store and return specific fields from the documents, sorted in a specific order. One of the examples also uses offset and limit. Note: You can improve the performance of order by queries by using secondary indexes. See [1]ORDER BY Queries for more information. * [2]Java - Order By * [3]Java - Order By with Offset and Limit * [4]dbshell The following example shows how to return the _id, firstName, lastName, and address.zipCode fields from documents in a store, sorting the documents by _id. It uses the [5]Query.select and [6]Query.orderBy methods. Note: The example sorts in the default ascending order.
