Create a Document using a Document Object This topic shows you how to create a new JSON document by calling methods on the Document object returned by the Connection.newDocument() method. To create a new document, you need to set fields and values in a Document object: 1. Create a new JSON document by calling the newDocument() method in the Connection class. 2. Specify the ID of the document with the setId() method. 3. Specify field names and their values with the set() or setArray() method. 4. Return the results in a Document object.
