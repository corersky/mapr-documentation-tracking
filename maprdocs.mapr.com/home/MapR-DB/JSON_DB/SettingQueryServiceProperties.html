Setting OJAI Distributed Query Service Properties OJAI queries may leverage the OJAI Distributed Query Service. This section describes how to modify OJAI Distributed Query Service property settings in your OJAI application. Call the following method, specifying the property name as the first argument and the value as the second. Prefix the OJAI Distributed Query Service property name with "ojai.mapr.drill". For example, the following call disables using secondary indexes when queries use the Query Service. * [1]Java query.setOption("ojai.mapr.drill.planner.enable_index_planning", false); See [2]Index Planning and Execution Configuration Options for a list of other OJAI Distributed Query Service properties.
