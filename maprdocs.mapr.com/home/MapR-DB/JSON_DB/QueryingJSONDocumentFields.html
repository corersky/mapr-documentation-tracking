JSON Document Query Conditions Using OJAI Syntax OJAI provides an interface for querying JSON documents. It includes syntax for specifying query conditiions using a JSON format. When writing an OJAI application, you can also apply query conditions by calling OJAI API methods, corresponding to specific operators. This section does not discuss this alternative.
