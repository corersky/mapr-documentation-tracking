Updating JSON Documents This section contains sample code that updates a JSON document in a MapR-DB JSON table. It also shows the MapR-DB Shell syntax for inserting documents. * [1]Java * [2]dbshell The following code is available at [3]OJAI_012_UpdateDocument.java. It does the following: 1. Finds a document using the [4]DocumentStore.find method 2. Creates a [5]DocumentMutation and updates a field 3.
