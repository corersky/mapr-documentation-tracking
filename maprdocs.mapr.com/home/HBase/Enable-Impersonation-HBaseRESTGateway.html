Enable Impersonation for HBase REST Gateway To enable HBase REST Gateway impersonation, configure all HBase servers to allow proxy users, then configure every REST Gateway to enable impersonation. To enable REST Gateway impersonation, add the following to the hbase-site.xml file for every REST gateway: hbase.rest.authentication.type kerberos hbase.rest.authentication.kerberos.principal HTTP/_HOST@HADOOP.LOCALDOMAIN hbase.rest.authentication.kerberos.keytab $KEYTAB Substitute the keytab for HTTP for $KEYTAB.
