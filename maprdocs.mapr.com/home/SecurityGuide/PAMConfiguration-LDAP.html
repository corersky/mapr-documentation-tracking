Configuring PAM to use LDAP To configure PAM with LDAP: 1. Verify that each MapR user ID has the auxiliary schema posixAccount. 2. Verify that each group ID has the auxiliary schema posixGroup. 3. Install the appropriate PAM packages: + On Ubuntu, sudo apt-get install libpam-ldapd + On Redhat/Centos, sudo yum install pam_ldap
