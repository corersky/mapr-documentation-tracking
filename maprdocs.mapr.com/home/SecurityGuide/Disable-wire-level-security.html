Disabling Wire-level Security To disable security features for your cluster: 1. If the cluster is running, [1]shut it down. 2. On all nodes, run the configure.sh script with the -unsecure option and specify the CLDB and ZooKeeper nodes. configure.sh -unsecure -C -Z -N 3. [2]Start the cluster.
