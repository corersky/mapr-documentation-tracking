Enabling Impersonation for the MapR Superuser Provides a procedure necessary to implement superuser impersonation. To enable impersonation in your MapR installation: 1. Open the following file in a text editor: /opt/mapr/hadoop/hadoop-2.7.0/etc/hadoop/core-site.xml 2. Add the following hadoop.proxyuser properties: hadoop.proxyuser.mapr.hosts * hadoop.proxyuser.mapr.groups * The hosts setting (*) allows the mapr superuser to connect from any host to impersonate a user. The groups setting (*) allows the mapr superuser to impersonate any user in any group. Note: Do not use anything other than a single asterisk here. Other parts of MapR ignore the values here and treat them as if each is set to a single asterisk. 3. Close the file, saving any changes that you made.
