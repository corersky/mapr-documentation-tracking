Integrating the Hive-on-Tez User Interface with Tez This topic describes how to integrate the Hive-on-Tez user interface with Tez. 1. Add the following entry to the /opt/mapr/tez/tez-/conf/tez-site.xml file, replacing : with the real host name. Use 9383 for the port. 9383 is the default Tomcat port for the Hive-on-Tez user interface. Enable Tez to use the Timeline Server for History Logging tez.history.logging.service.class org.apache.tez.dag.history.logging.ats.ATSHistoryLoggingService URL for where the Tez UI is hosted tez.tez-ui.history-url.base http://:/tez-ui/
