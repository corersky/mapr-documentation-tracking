Configure Encryption without Authentication Complete the following steps on each node where HiveServer2 is installed: 1. In hive-site.xml file, set the following properties: Property Value hive.server2.use.SSL true hive.server2.ssl.keystore hive.server2.ssl.keystore.password Warning: If you specify the password in the hive-site.xml file, protect the file with the appropriate file permissions. HiveServer2 automatically prompts for the keystore password during startup when no password is stored in the hive-site.xml file. hive.server2.use.ssl true enable/disable SSL communication hive.server2.ssl.keystore path to keystore file hive.server2.ssl.keystore.password keystore password 2. Restart HiveServer2 to apply these changes. maprcli node services -name hs2 -action restart -nodes
