Using JDBC or Beeline to Connect to HiveServer2 The method that HiveServer2 clients use to connect to HiveServer2 is based on the HiveServer2 Authentication method and the type of client. Using ODBC to Connect to HiveServer2 For details on how to install and use ODBC to connect to Hive, see [1]Using ODBC to Connect to HiveServer2. Note: The client must have a Kerberos ticket.
