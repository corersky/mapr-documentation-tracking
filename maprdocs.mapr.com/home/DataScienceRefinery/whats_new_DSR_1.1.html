What's New in MapR Data Science Refinery 1.1 Provides a summary of the new functionality in MapR Data Science Refinery 1.1 Data Science Refinery 1.1 includes the following new features in Zeppelin on MapR: * Support for the Spark interpreter, configured to launch Spark jobs in YARN client mode * Enhancements in installing custom Python environments for the Livy and Spark interpreters * Improvements in launching multiple Zeppelin containers on the same host
