MapR-DB Connectors for Apache Spark This section describes the MapR-DB connectors that you can use with Apache Spark. [1]Apache Spark is a software framework that is used to process data in memory in a distributed manner. Spark is replacing MapReduce in many use cases. The MapR-DB Connectors for Spark enable users to write applications that access MapR-DB JSON and Binary tables. References 1. http://spark.apache.org/
