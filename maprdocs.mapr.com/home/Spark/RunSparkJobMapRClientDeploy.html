Running Jobs as mapr in Client Deploy Mode This topic describes how to run jobs with Apache Spark on Apache Mesos as user 'mapr' in client deploy mode. The following example submits a job running the SparkPi sample code: /opt/mapr/spark/spark-2.1.0/bin/spark-submit \ --class org.apache.spark.examples.SparkPi \ --master mesos://:5050 \ --deploy-mode client \ /opt/mapr/spark/spark-2.1.0/examples/jars/spark-examples_2.11-2.1.0-mapr-SNA PSHOT.jar 100
