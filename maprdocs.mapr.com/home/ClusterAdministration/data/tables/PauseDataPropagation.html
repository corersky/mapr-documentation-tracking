Pausing Data Propagation You can pause data propagation using MCS and the CLI. Pausing Data Propagation Using the MapR Control System 1. Log in to MCS and go to the Change Data Capture tab in the [1]table information page. 2. Select the change log(s) to pause. Selecting the checkbox next to a change log makes the Actions dropdown menu available. 3. Select Pause Data Propagation from the Actions dropdown menu to display the Pause Data Capture confirmation dialog. 4. Review the change log(s) to pause and click Pause Data Capture. Pausing Data Propagation Using the CLI or REST API The basic command to pause change data propagation is: maprcli table changelog pause For complete reference, see [2]table changelog pause.
