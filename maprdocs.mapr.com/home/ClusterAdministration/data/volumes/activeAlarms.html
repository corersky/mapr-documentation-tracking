Viewing Active Volume Alarms You can view volume alarms in MCS and using the CLI. Viewing Active Volume Alarms in the MapR Control System Log in to MCS and: * Click Data > Volumes to view all active volume alarms in the Active Alarms pane. * Go to the Summary tab in the [1]volume information page to view the recent and active alarms for the selected volume in the Alarms pane. * Click [alarmSummary.png] (in the top navigation bar) and select Volume Alarms from the dropdown menu in the All alarms pane to view all the active volume alarms. * Click Overview and select Volume Alarms from the dropdown menu in the Active Alarms pane to view all active volume alarms. You can: * [2]View information related to the alarm. * [3]Dismiss an alarm. * [4]Mute an alarm. See [5]Volume Alarms for more information on the volume alarms. Retrieving Active Volume Alarms Using the CLI or REST API The basic command to retrieve node alarms is: maprcli alarm list -cluster -type volume For complete reference information, see [6]alarm list.
