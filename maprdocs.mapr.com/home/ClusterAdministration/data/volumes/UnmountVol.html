Unmounting one or more Volumes You can unmount multiple volumes using MCS and the CLI. To unmount volumes under a directory, you must have read/write permissions on the directory. Unmounting one or more Volumes Using the MapR Control System To unmount one or more volumes, in the Summary tab under Data > Volumes: 1. Select the volumes to unmount from the list of volumes in the Volumes pane. 2. Select Unmount Volume(s) from the Actions dropdown menu. The Unmount Volume(s) confirmation dialog displays. Note: Only mounted volumes can be unmounted. Volumes that are currently not mounted, if selected, cannot be unmounted. 3. Verify list of selected volumes. If necessary, click to remove a volume from the list of volumes to unmount. 4. Click Unmount Volumes to unmount the selected volumes. Unmounting one or more Volumes Using the CLI or REST API The basic command to unmount the volume is: maprcli volume unmount -name For complete reference information including all available options, see [1]volume unmount.
