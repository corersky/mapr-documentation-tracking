Customizing the List of Columns/Fields When viewing the list of volumes/nodes, you can customize the columns displayed in MCS and the fields returned in the CLI. Customizing the Columns in the MapR Control System 1. Log in to MCS and click: + Data > Volumes to customize columns displayed in the Volumes pane. + Nodes to customize columns displayed in the Nodes pane. 2. Select Customize Columns from the [gearDropDownMenu.png] dropdown menu. In the Customize Columns dialog, the: + Available list displays the columns available, but not displayed. + Selected list displays the columns currently displayed in the pane. 3. Select the columns from the: a. Available list of columns and click [rightArrow.png] to move selection to Selected columns (for display). b. Selected list of columns and click [leftArrow.png] to remove selected columns from displaying. 4. (Optional) Click [upArrow.png] and/or down [downArrow.png] arrows to sort the order of columns. 5. Click Save Changes for the customization to take effect. Customizing the Fields Using the CLI or REST API Use the -column parameter with the maprcli command to view specific fields in the list. For example: * To view the health of the nodes and services installed on the nodes being retrieved, run the following command: maprcli volume list -column service,health For complete reference information, see [1]node list command. * To view the volume name for the list of volumes being retrieved, run the following command: maprcli volume list -column volumename For complete reference information, see [2]volume list command.
