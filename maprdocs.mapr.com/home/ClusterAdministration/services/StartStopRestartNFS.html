Starting, Stopping, and Restarting MapR NFS You can start, stop, and restart NFS using MCS and the CLI. Starting, Stopping, and Restarting MapR NFS Using the MapR Control System See: * [1]Starting the Services on the Cluster Using the MapR Control System * [2]Stopping a Service on the Cluster Using the MapR Control System * [3]Restarting the Services on the Cluster Using the MapR Control System Starting, Stopping, and Restarting MapR NFS Using the CLI NFSv3 Server The command to stop, start, or restart MapR NFSv3 server is: maprcli node services -nodes -nfs stop|start|restart For complete reference, see [4]node services. Note: When NFS server is stopped, the VIPs associated with the server are released and CLDB attempts to reassign the VIPs to other available NFS servers.
