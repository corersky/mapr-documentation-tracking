Stopping the Services You can stop one or more services using MCS and the CLI. Stopping the Services Running on the Nodes Using the MapR Control System To stop the services running on the nodes: 1. Log in to MCS and click Nodes to display the Nodes page. 2. Select the nodes from the list of nodes in the Nodes pane and click Manage Services to display the Manage Services window. 3. Choose the Stop radio button for the services you wish to stop on the selected nodes and click Save. Stopping the Services on a Node Using the MapR Control System To stop one or more services running on a node: 1. Go to the Summary tab in the [1]node information page. 2. Select the services to stop in the Services pane. 3. Click Stop Service(s). The Stop Service(s) confirmation dialog displays. 4. Verify the list of services to stop and click Stop Service. Stopping a Service on the Cluster Using the MapR Control System 1. Log in to MCS and under Services. 2. Click [serviceStop.png] associated with the service to stop. The Stop Service confirmation dialog displays. 3. Verify the list of nodes on which to stop the service and click Stop Service. Stopping the Services Using the CLI or REST API The basic command to stop a service on a node is: maprcli node services -nodes -name -action stop For complete reference information, see [2]node services.
