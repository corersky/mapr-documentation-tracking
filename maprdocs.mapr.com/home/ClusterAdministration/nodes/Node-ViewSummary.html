Viewing Node Summary You can view a summary of the alarms on a node, services on a node, and disks on a node through MCS and using the CLI. Viewing Node Summary Using the MapR Control System Log in to MCS and go to the [1]node information page. The page with information on the node displays and the Summary tab displays by default. The Summary tab contains panes for node-specific: * Alarms — displays active and recent alarms on the node. See [2]Viewing Active Node Alarms for more information. * Services — displays services running on the node. See [3]Viewing the Services Running on a Node Using the MapR Control System for more information. * Disks — displays information on the disks on the node. See [4]Viewing the List of Disks for more information. Viewing Node Summary Using the CLI The basic command to retrieve a summary of the disks on a node is: maprcli node metrics -nodes -columns DISKS For complete reference information, see [5]node metrics.
