Retrieving Balancer Status Run the maprcli volume balancinginfo command to retrieve the status of a currently running or scheduled balancer activity. The basic command to retrieve the status of volume balancer is: /opt/mapr/bin/maprcli volume balancinginfo -name For more information, see [1]volume balancinginfo.
