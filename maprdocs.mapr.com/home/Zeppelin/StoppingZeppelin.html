Stopping Your Zeppelin Container If you no longer need to use your Zeppelin container, you can stop the running image. 1. Determine the container id of your docker instance by listing all running docker containers: docker ps The output includes the following fields: CONTAINER ID IMAGE 23b60c2720dd maprtech/data-science-refinery:v1.2_6.0.1_5.0.0_centos7 2. Stop the running container specifying the revealed container id: docker stop 23b60c2720dd
