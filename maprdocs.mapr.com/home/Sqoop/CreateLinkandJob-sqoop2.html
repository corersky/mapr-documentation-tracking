Create Links and Jobs Use connectors to create links to data sources and then create jobs to execute the data transfer. 1. Create a link to each data source using the connector name. sqoop:000> create link --connector For example, to create a link with the kite-connector: sqoop:000> create link --connector kite-connector 2. Use link names to create a job. sqoop:000> create job --from --to For more information about using connectors, see the Sqoop2 connector documentation for [1]Sqoop 1.99.6 and [2]Sqoop 1.99.7. Note: Prior to using an FTP or an SFTP connector, verify that the associated FTP or SFTP server is configured. References 1. https://sqoop.apache.org/docs/1.99.6/Connectors.html 2. https://sqoop.apache.org/docs/1.99.7/user/Connectors.html
