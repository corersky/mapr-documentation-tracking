Kafka REST 2.0.1: GET /streams/{stream: string}/topics Retrieves a list of topics in a given stream. Description Stream names contain characters such as backslashes (\) and colons (:) and, therefore, should be encoded. CAPTION: Table 1. Parameters Parameters Description stream_name (string) The name of the stream. Syntax http://:8082/streams//topics Request Example curl -X GET http://localhost:8082/streams/%2Fstreaming_data%2Fstream/topics Response Example [ "/streaming_data/stream:testtopic1", "/streaming_data/stream:testtopic2" ]
