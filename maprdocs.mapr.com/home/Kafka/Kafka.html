Supported Kafka Connect/REST versions MapR-ES Tool MEP Release Kafka version Kafka REST/Connect 4.0.0 5.0 MEP release 1.0 Kafka REST/Connect 2.0.1 2.x, 3.x, and 4.x MEP release 0.9 Kafka REST/Connect 4.0.0: * [1]Kafka REST Proxy 4.0.0 for MapR-ES is based on Kafka 1.0. This tool is used as a RESTful interface to MapR-ES. This tool is available as of MEP 5.x. * [2]Kafka Connect 4.0.0 for MapR-ES is based on Kafka 1.0. This tool is used to stream data between MapR-ES and other storage systems. This tool is available as of MEP 5.x. Kafka REST/Connect 2.0.1: * [3]Kafka REST Proxy 2.0.1 for MapR-ES is based on Kafka 0.9. This tool is used as a RESTful interface to MapR-ES. This tool is available as of MEP 2.x, 3.x, and 4.x. * [4]Kafka Connect 2.0.1 for MapR-ES is based on Kafka 0.9. This tool is used to stream data between MapR-ES, Apache Kafka, and other storage systems. This tool is available as of MEP 2.x, 3.x, and 4.x. MapR-ES Clients MapR-ES Client applications can be developed for MapR-ES (as of MapR 5.2.1 with MEP 3.0). The MapR-ES clients are based on distributions of librdkafka that works with MapR-ES. * MapR-ES C Client - Used to develop MapR-ES applications in C. See [5]MapR-ES C Applications * MapR-ES Java Client - Used to develop MapR-ES applications in Java. See [6]MapR-ES Java Applications * MapR-ES Python Client - Used to develop MapR-ES applications in Python. This client is available as of MapR 5.2.1 with MEP 3.0.
