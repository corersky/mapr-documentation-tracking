Configuring the Parquet Block Size As of Drill 1.12, the default value for the store.parquet.block-size parameter is 268435456 (256 MB), the same size as MapR-FS chunk sizes. In previous versions of Drill, the default value was 536870912 (512 MB). If you change the MapR-FS chunk size, change the Parquet block size to match using the [1]ALTER SYSTEM or [2]SET commands, as shown: ALTER SYSTEM SET `store.parquet.block-size` = ; [ALTER SESSION] SET `store.parquet.block-size` = Alternatively, you can override the default setting in the /conf/drill-override.conf file, as shown: drill.exec: { ... options.store.parquet.block-size = 268435456 } For information about setting the MapR-FS chunk size, see [3]Setting Chunk Size.
