New in This Release Sentry 1.7.0-1703 introduces the following enhancements or MapR platform-specific behavior changes: * Sentry integration with Hive authorization framework v2 * Create simple shell for Sentry * Upgrade Hive plugin v2 for Hive 2.1.1 Patches This MapR release includes the following patches on the base Apache release.
