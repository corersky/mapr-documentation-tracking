Commit Date (YYYY-MM-DD) Comment ae79594 2016-05-17 MAPR-23302: ListBucketing tables with uppercase skew values can now be created. 1878e90 2016-04-25 MAPR-23153: Hive will no longer create scratch dir with incorrect permissions when a CTAS query is run.
