MEP Package Names This topic provides a link to the packages associated with MapR Expansion Packs. To view the package names for a MEP: 1. Use a browser to navigate to [1]http://package.mapr.com/releases/MEP/. A list of MEP links is displayed. 2. Click the link for your MEP. A list of operating system links is displayed. 3. Click the link for your operating system. The list of package names is displayed. References 1. http://package.mapr.com/releases/MEP/
