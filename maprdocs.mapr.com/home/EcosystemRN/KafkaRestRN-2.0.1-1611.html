Version 2.0.1 Release Date December 9, 2016 MapR Version Compatibility See [1]Ecosystem Support Matrix Source on GitHub n/a GitHub Release Tag 2.0.1-mapr-1611 Package Names See [2]Package Names for MapR Expansion Packs (MEPs) New in the Release This release is the first MapR release of the Kafka REST Proxy for MapR-ES. Known Issues and Limitations The following are either known issues or limitations. * Impersonation is not supported for Kafka REST Proxy for MapR-ES. Kafka REST is run as mapr user which means that you must explicitly give mapr user permissions to consume and produce from/to streams. For example, the consumeperm parameter must be set to mapr user to be able to consume and the produceperm parameter must be set to mapr user to be able to produce.
