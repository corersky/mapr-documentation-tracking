Using Flume Start a Flume Agent You can start a Flume agent with the following command: /opt/mapr/flume/flume-/bin/flume-ng agent --conf-file -name Note: To send debugging output to the console, add the flag -Dflume.root.logger=DEBUG,console. To send debugging output to a log file, add the flag -Dflume.root.logger=DEBUG,LOGFILE. You can configure the location of this logfile in the flume-/conf/log4j.properties file.
