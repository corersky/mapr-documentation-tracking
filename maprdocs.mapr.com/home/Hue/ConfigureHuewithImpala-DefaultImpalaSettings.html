Default Impala Settings in hue.ini The default version of the impala section of the hue.ini file is shown below. #################################################################### # Settings to configure Impala #################################################################### [impala] # Host of the Impala Server (one of the Impalad) ## server_host=localhost # Port of the Impala Server ## server_port=21050 # Turn on/off impersonation mechanism when talking to Impala ## impersonation_enabled=False
