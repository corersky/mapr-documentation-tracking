Using a Non-Default Authentication Method The default authentication method is PAM. To edit the authentication method used for the Hue interface, complete the following steps: 1. Set the backend property equal to your selected authentication method. For example, to use Hue's user authentication, select desktop.auth.backend.AllowFirstUserDjangoBackend. 2. If you choose not to use PAM, comment the pam_service property. Example hue.ini configued to use PAM Authentication [[auth]] # Authentication backend.
