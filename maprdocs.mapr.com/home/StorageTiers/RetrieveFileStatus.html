Retrieving Status of File-level Tiering Operation Using the CLI You can retieve the status of a file-level tiering operation using the CLI. For information on volume-level tiering operation, see [1]Retrieving the Status of a Volume-level Tiering Operation. Run the following command to check the status of an active or completed offload, abort, and/or recall operation. maprcli file tierjobstatus -name -json See [2]Output for more information.
