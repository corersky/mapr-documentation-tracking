YARN Application Requirements The following tables include the minimal node requirements for building and running YARN applications. CAPTION: Node Requirement Method(s) to Meet Requirement A connection to the MapR cluster. Select one of the following options: * Install and configure the MapR client. * Install the PACC and run an application container. For more information, see [1]Connect to the Cluster. Hadoop libraries are configured as an application dependency. When you compile the application, use the Maven Repository to determine the dependencies.
