hdfsBuilderSetNameNode() Supported by libMapRClient for hadoop-2.x Specifies a CLDB node for an hdfsBuilder structure. Signature void hdfsBuilderSetNameNode(struct hdfsBuilder *bld, const char *nn) Parameters Parameter Description bld An hdfsBuilder structure. This value cannot be NULL. nn The hostname or IP address of a name node. Use NULL to connect to the local filesystem. Use default to connect to the first MapR Filesystem cluster that is listed in the MAPR_HOME/conf/mapr-clusters.conf file on the client.
