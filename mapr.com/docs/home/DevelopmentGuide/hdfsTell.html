hdfsTell() Supported by libMapRClient for hadoop-2.x Gets the current offset in the file in bytes. Signature tOffset hdfsTell(hdfsFS fs, hdfsFile file) Parameters Parameter Description fs The handle of the filesystem where the file is located. Obtain this handle with one of the hdfsConnect() APIs. file The file handle. Obtain this handle with one of the hdfsOpenFile() APIs. Return Value Returns the current offset in bytes, or -1 on error. Check errno for error codes and meanings.
