hdfsAvailable() Supported by libMapRClient for hadoop-2.x Returns the number of bytes that can be read from an input stream without blocking. This number is simply the size of the file in bytes. Signature int hdfsAvailable(hdfsFS fs, hdfsFile file) Parameters CAPTION: Parameter Description fs The handle of the filesystem where the file is located. Obtain this handle with one of the hdfsConnect() APIs. file The file handle. Obtain this handle with one of the hdfsOpenFile() APIs. Return Value Returns the size of the file in bytes, -1 on error. Check errno for error codes and meanings. errno is set to EINVAL if the arguments provided are invalid.
