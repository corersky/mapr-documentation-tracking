hdfsGetCapacity() Supported by libMapRClient for hadoop-2.x Returns the capacity in bytes of the connected filesystem. Signature tOffset hdfsGetCapacity(hdfsFS fs) Parameters CAPTION: Parameter Description fs The filesystem handle. Obtain this handle by calling one of the hdfsConnect() APIs. Return Value Returns the capacity in bytes of the connected filesystem, -1 on error. Check errno for error codes and meanings. errno can be set to EINVAL in case of error.
