Integrating the MapR GitHub and Maven Repositories This topic provides instructions for cloning the GitHub and Maven repositories for a MapR open source project into your Eclipse IDE. Integrating Git 1. Open the Git Repository perspective by selecting Window>Open Perspective>Other... then choosing Git Repository Exploring. 2. From the Git Repository perspective, click the [27199753.png] button to display the Clone Git Repository dialog. [27199750.png] 3. From a web browser, navigate to the MapR [1]repository, then select the project you want to clone. 4. Copy the git URI from the project page to your clipboard by clicking the [27199752.png] button. 5. In the Clone Git Repository dialog, paste the git URI into the URI: field, then click Next. Eclipse will connect to github and download the repository metadata, then display a list of branches. 6. Select the branches you wish to clone, then click Next. 7. Configure the destination directory, then click Finish. Eclipse downloads the project from github and adds it to your view. Integrating Maven 1. Start a new Maven project, or convert your current project into a Maven project if necessary. 2. Select Window>Show View>Package Explorer to show your current Maven project. 3. Add the following lines to your project's pom.xml file: mapr-releases http://repository.mapr.com/maven/ false true 4. In a browser, navigate to the MapR [2]Maven Repository and search for the Maven artifact your project depends on. You can also [3]browse the repository. 5. In the Package Explorer, right-click your project and select Maven>Add Dependency. [27199751.png] 6. Enter the groupId, artifactId, and version values for the dependency, then click OK. 7. Refresh the workspace by pressing F5. Your Maven dependencies download automatically.
