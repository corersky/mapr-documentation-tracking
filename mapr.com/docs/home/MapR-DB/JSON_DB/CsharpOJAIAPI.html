C# OJAI Client Classes and Methods This topic lists and describes the classes supported by the C# OJAI client and provides a link to document pages that describe the methods in each class. Class Name Description [1]Value Encapsulates the value of a field, scalar, or complex in an OJAI document. [2]OjaiDocument Provides the primary, DOM-based interface for inspecting OJAI documents [3]OjaiDocumentStream/QueryResult Encapsulates the result set of an OJAI query. [4]OjaiDocumentMutation Encapsulates a mutation to an existing OJAI document in a store. [5]Query Encapsulates an OJAI query. [6]QueryCondition Encapsulates a query condition; similar to a SQL where clause. [7]ConnectionFactory Provides a logical connection to an OJAI data source: for example, MapR Database JSON. [8]MapRDBConnection Provides a connection handler, which enables you to get and check connections. [9]OjaiDocumentStore Encapsulates a store, typically persistent, of OJAI documents: for example, MapR Database JSON tables. [10]OTime Encapsulates the OJAI TIME type. [11]OTimestamp Encapsulates the OJAI TIMESTAMP type. [12]ODate Encapsulates the OJAI DATE type. [13]OInterval Encapsulates the OJAI INTERVAL type. See the [14]C# OJAI Client API for details about each class, including the methods available in each class.
