Querying with Conditions The examples in this section query a document store and return documents that have specific conditions. For more information about how to specify query conditions in OJAI, see [1]Query Conditions in OJAI Applications. Note: You can improve the performance of queries with conditions by using secondary indexes. See [2]Queries that Benefit from Secondary Indexes for more details. * [3]Java - OJAI QueryCondition Object * [4]Java - OJAI Query Condition in JSON Format * [5]dbshell The following example shows how to return all documents from a store where address.zipCode equals 95196, using the [6]Query.where method.
