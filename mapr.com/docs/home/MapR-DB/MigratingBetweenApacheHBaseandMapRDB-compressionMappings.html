Compression Mappings MapR Database binary tables support the LZ4, LZF, and ZLIB compression algorithms. When you create a MapR Database binary table with the Apache HBase API or the HBase shell and specify the LZ4, LZO, or SNAPPY compression algorithms, the table uses the LZ4 compression algorithm. When you describe a MapR Database binary table's schema through the HBase API, the LZ4 and OLDLZF compression algorithms map to the LZ4 compression algorithm.
