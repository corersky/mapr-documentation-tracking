libMapRClient C APIs This section provides the MapR-DB libMapRClient C API library. This library is MapR's extension of the libhbase C API library. The libMapRClient header files are in this directory: /opt/mapr/include/hbase The libMapRClient API implements functions in addition to the functions in the libhbase API. Note: Your applications need include only the hbase.h header file. The header files are provided for display purposes.
