Impala 1.4.1-1501 Release Notes CAPTION: Version 1.4.1 Release Date January 14, 2015 Source on GitHub [1]https://github.com/mapr/impala GitHub Release Tag 1.4.1-mapr-1501 MapR Version Interoperability See [2]Impala Support Matrix and [3]Ecosystem Support Matrix (Pre-5.2 releases). This release of Impala includes some updated package files that do not affect Impala performance or functionality. Important: * Impala on MapR version 4.0.1 requires mapr-patch-4.0.1.27334.GA-28668.x86_64.rpm. Contact support to obtain the patch. If Oozie is running on the cluster, after you apply the patch you must regenerate the Oozie WAR file to pick up the new binary. * Impala on MapR version 4.0.2 does not require a patch. * This release of Impala only works with Sentry in file-based storage mode.
