The MapR Kafka Streams is included in MEP repositories beginning with MEP-6.0.0. Kafka Streams is a Java library for building streaming applications and microservices, specifically ones that transform input MapR Event Store For Apache Kafka topics into output MapR Event Store For Apache Kafka topics.
