Hue 3.12.0-1901 (MEP 4.1.3 and MEP 5.0.2) Release Notes This section provides reference information, including new features, patches, known issues, and limitations for Hue 3.12.0-1901. The notes below relate specifically to the MapR Distribution for Apache Hadoop. You can find additional information on this [1]changelog or the [2]Hue homepage. These release notes contain only MapR-specific information and are not necessarily cumulative in nature. For information about how to use the release notes, see [3]Ecosystem Component Release Notes.
