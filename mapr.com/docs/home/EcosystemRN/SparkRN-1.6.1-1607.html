Source on GitHub [4]https://github.com/mapr/spark/tree/1.6.1-mapr-1607 Package Names The following packages are associated with this release: * mapr-spark-1.6.1.201607242143-1.noarch.rpm * mapr-spark_1.6.1.201607242143_all.deb * mapr-spark-historyserver-1.6.1.201607242143-1.noarch.rpm * mapr-spark-historyserver_1.6.1.201607242143_all.deb * mapr-spark-master-1.6.1.201607242143-1.noarch.rpm * mapr-spark-master_1.6.1.201607242143_all.deb New in This Release This release of Apache Spark includes the following behavior change that is specific to MapR: Poll Time for Consuming MapR Event Store For Apache Kafka When Spark consumes MapR Event Store For Apache Kafka messages, the default poll time is 1000 milliseconds. Previously, the default was 100 milliseconds. Important Notes If you want to integrate Spark 1.6.1-1607 with MapR Event Store For Apache Kafka, you must install the Kafka 0.9.0-1607 package.
