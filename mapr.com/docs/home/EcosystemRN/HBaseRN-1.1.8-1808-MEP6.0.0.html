HBase 1.1.8-1808 (MEP 6.0.0) Release Notes The notes below relate to the MapR Converged Data Platform. Version 1.1.8 Release Date September 2018 MapR Version Interoperability See [1]MEP Components and OS Support. Source on GitHub [2]https://github.com/mapr/hbase/tree/rel/1.1.8 Maven Artifacts [3]http://repository.mapr.com/maven/ Package Names Navigate to [4]https://package.mapr.com/releases/MEP/ and select your MEP and OS to view the list of package names. New in this Release * HBase Master and HBase Regionserver are removed for MapR 6.0.0 and MEP 4.0.0. The Thrift and REST server packages are still available to support MapR Database binary tables. * MapR-SASL authentication, encryption, and impersonation for HBase Thrift Gateway are enabled by default on secure clusters. * On a secure cluster, by default, HBase REST and HBase Thrift read the ssl-client.xml file and configure SSL using this file. * PAM authentication, encryption, and impersonation for HBase REST are enabled by default on secure clusters. Note: HBase Master and HBase Regionserver are still available for previous MEP releases.
