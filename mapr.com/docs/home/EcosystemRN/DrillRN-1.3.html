Drill 1.3.0 Release Notes (Developer Preview) The following release notes are for the 1.3.0 developer preview of the MapR distribution of Apache Drill. Release notes for prior releases are posted on the Apache Drill web site. Version 1.3.0 Developer Preview Release Date November 23, 2015 MapR Version Interoperability See [1]Interoperability Matrix and [2]Drill Support Matrix Noteworthy New Features This release introduces the following enhancements: * Upgrade to the latest version of the Apache Parquet library (See [3]DRILL-4070 and [4]http://drill.apache.org/docs/partition-by-clause/) * Performance improvements for Limit 0 and Limit 1 queries * Hive table query planning improvements * Partition pruning enhancements * Support for CSV header parsing * Improved query/operators profiles * Capability to reset the SYSTEM and SESSION settings to their defaults * Merge Join improvements * [5]Additional bug fixes and enhancements Known Issue The MapR 1.3 developer preview of Drill has the following known issue: * [6]DRILL-4070 - You must change the metadata of Parquet files generated by Drill 1.1 and 1.2 for the files to work optimally with Drill 1.3.0. See http://drill.apache.org/docs/partition-by-clause/.
