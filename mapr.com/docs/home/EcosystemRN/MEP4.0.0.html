CAPTION: Release Date November 2017 Repository Location https://package.mapr.com/releases/MEP/^1 ^1The MEP- directory can be represented by a 2-digit number or a 3-digit number (for example, MEP-2.0 or MEP-2.0.0). See [1]Understanding Two-Digit and Three-Digit MEPs. To understand which MapR Core versions can use this MEP, see [2]MEP Support by MapR Core Version. Release Note Naming Convention The release note naming convention is based on version number and release date. For Hive 2.3.3-1808, 2.3.3 refers to the Hive version number, and 1808 typically indicates an August 2018 release, but last-minute changes in the release date can result in a slight mismatch between the naming convention and the actual release.
