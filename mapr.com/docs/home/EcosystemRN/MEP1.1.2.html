MapR Expansion Pack 1.1.2 Release Notes This topic contains information about the components included with the MapR Expansion Pack 1.1.2. Release Date April 2017 Repository Location http://package.mapr.com/releases/MEP/^1 ^1The MEP- directory can be represented by a 2-digit number or a 3-digit number (for example, MEP-2.0 or MEP-2.0.0). The three-digit MEP directory contains a fixed MEP version with patches. Use this MEP version if you prefer to do manual installs and do not require patch updates. The two-digit MEP directory contains the latest MEP and patches and is continuously updated. The MapR installer uses the two-digit MEP directory to make new patches available automatically without the need for system reconfiguration. Note: The release note naming convention is based on version number and release date. For Hive 2.1.1-1803, 2.1.1 refers to the Hive version number and 1803 typically indicates a March 2018 release but last-minute changes in the release date can result in a slight mismatch between the naming convention and the actual release.
