Package Names for MapR Ecosystem Packs (MEPs) This page describes how to view the the package names for each MapR Ecosystem Pack (MEP) release. To view the package names for a MEP: 1. Use a browser to navigate to [1]https://package.mapr.com/releases/MEP/. A list of MEP links is displayed. 2. Click the link for your MEP. A list of operating system links is displayed. 3. Click the link for your operating system. The list of package names is displayed. For more information about the supported MEPs, see [2]MEP Components and OS Support. For information about packages and dependencies for MapR 6.0, see [3]Packages and Dependencies for MapR Software.
