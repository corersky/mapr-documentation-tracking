MapR Monitoring Components - MEP 6.0.0 Release Notes The notes below relate specifically to the MapR Converged Data Platform. These release notes contain only MapR-specific information and are not necessarily cumulative in nature. For information about how to use the release notes, see [1]Ecosystem Component Release Notes. CAPTION: Version The MEP 6.0.0 release contains the following monitoring component versions: * Collectd 5.8.0 * Fluentd 1.1.2 * Opentsdb 2.4.0 * Elasticsearch 6.2.3 * Grafana 4.6.1 * Kibana 6.2.3 Release Date September 2018 MapR Version Interoperability [2]Component Versions for Released MEPs Package Names [3]Package Names for MapR Ecosystem Packs (MEPs) New in This Release MEP 6.0.0 includes new versions of the following MapR Monitoring components: Collectd, Fluentd, Elasticsearch, and Kibana. To compare MapR Monitoring component versions, see [4]Component Versions for Released MEPs. If you are upgrading from another release to MapR 6.1, you might need to perform extra steps to convert or preserve the Elasticsearch and Kibana indexes before upgrading.
