Oozie Support Matrix This matrix shows the interoperability between Oozie and other ecosystem products. This page shows the versions of Oozie that work with other ecosystem components and it applies to MapR versions 5.1 and below. For more recent versions of MapR, see the [1]MEP Components and OS Support.
