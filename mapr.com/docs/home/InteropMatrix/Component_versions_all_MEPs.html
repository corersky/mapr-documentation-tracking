Component Versions for Released MEPs This page shows the ecosystem component versions included in published MapR Ecosystem Packs (MEPs), allowing you to compare the versions across MEPs. Components Supported by MEPs 1.x - 6.x The following table shows the published MEP series numbers (6.x, 5.x, 4.x, and so on) and the MapR core version on which each series is supported. Each MEP series can consist of several MEPs. In most cases, the MEPs within a series support the same version of a component. For example, all MEPs in the MEP 5.x series (MEPs 5.0.1 and 5.0.0) support Drill 1.13. However, there are exceptions. When multiple component versions are supported within a MEP series, the table shows the latest supported version and includes an asterisk (*). To see the other versions supported on MEPs in a series, navigate to [1]MEP Components and OS Support. Then select the link that corresponds to the MEP for which you need information. A long dash (—) indicates that the component is not present in the MEP series.
