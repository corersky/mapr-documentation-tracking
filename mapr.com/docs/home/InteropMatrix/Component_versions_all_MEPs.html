Component Versions for Released MEPs This page shows the ecosystem component versions included in published MapR Ecosystem Packs (MEPs), allowing you to compare the versions across MEPs. Components Supported by MEPs 1.x - 6.x The following table shows the latest supported version of each component in a MEP series. The table reflects the component versions at release time and does not include patch versions issued between releases. The table shows the published MEP series numbers (6.x, 5.x, 4.x, and so on) and the MapR core version that each MEP supports. Each MEP series can consist of several MEPs. Often, the MEPs within a series support the same version of a component, but not always. When a MEP series includes different component versions, the table shows the latest supported version and includes an asterisk (*). To see all the versions supported on MEPs in a series, navigate to [1]MEP Components and OS Support. Then select the MEP for which you need information. A long dash (—) indicates that the component is not present in the MEP series.
