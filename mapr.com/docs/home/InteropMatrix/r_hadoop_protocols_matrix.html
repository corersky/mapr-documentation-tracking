Hadoop Protocol Versions for MapR Software Shows the Hadoop RPC protocol version and compatible MapR client versions for each MapR release. Each MapR Cluster version is associated with a Hadoop RPC protocol version. The JobTrackers or ResourceManagers in a given cluster accept only the jobs submitted from clients with a compatible protocol version.
