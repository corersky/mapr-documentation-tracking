Read and Write MapR Event Store For Apache Kafka Messages With Flume Flume uses agent files to define the data flow. To read or write MapR Event Store For Apache Kafka data, you configure Flume agent files with Flume sources, channels, and sinks. When you read MapR Event Store For Apache Kafka topics, you use a Kafka source or a Kafka channel. When you write to a MapR Event Store For Apache Kafka topic, you can use a Kafka channel or a Kafka sink.
