Read and Write MapR-ES Messages With Flume Flume uses agent files to define the data flow. To read or write MapR-ES data, you configure Flume agent files with Flume sources, channels, and sinks. When you read MapR-ES topics, you use a Kafka source or a Kafka channel. When you write to a MapR-ES topic, you can use a Kafka channel or a Kafka sink.
