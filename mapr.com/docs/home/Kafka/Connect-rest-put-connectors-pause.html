PUT /connectors/(string:name)/pause Pauses the connector and its tasks, which stops message processing until the connector is resumed. Syntax http://:8083/connectors//pause Note: This call is asynchronous and the tasks will not transition to PAUSED state at the same time. Request Example PUT /connectors/hdfs-sink-connector/pause HTTP/1.1 Host: connect.example.com Response Example HTTP/1.1 202 Accepted
