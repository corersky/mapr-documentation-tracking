JDBC Driver Kafka Connect for MapR Event Store For Apache Kafka provides a JDBC driver jar along with the connector configuration. The JDBC driver (kafka-connect-jdbc) is set up by specifying the CLASSPATH variable. See [1]Installing MapR Event Store For Apache Kafka Tools.
