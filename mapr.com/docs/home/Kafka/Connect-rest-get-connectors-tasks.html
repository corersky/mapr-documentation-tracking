GET /connectors/(string:name)/tasks Gets a list of tasks currently running for the connector. Description CAPTION: Table 1. Parameters Parameters Description tasks (array) List of active task configurations created by the connector. tasks[i].id (string) ID of the task. tasks[i].id.connector (string) Name of the connector that the task belongs to. tasks[i].id.task (int) Task ID within the connector. tasks[i].config (map) Configuration parameters for the task.
