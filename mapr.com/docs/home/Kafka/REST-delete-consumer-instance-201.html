Kafka REST 2.0.1: DELETE /consumers/{group: string}/instances/{instance: string} Destroys the consumer instance. Description The request must be made to the specific REST proxy instance holding the consumer instance. CAPTION: Table 1. Parameters Parameters Description group_name (string) The name of the consumer group. instance (string) The ID of the consumer instance Syntax http://:8082/topics//instances/ Request Example DELETE /consumers/testgroup/instances/my_consumer HTTP/1.1 curl -X DELETE \ http://localhost:8082/consumers/my_binary_consumer/instances/rest-consumer-11561 681-8ba5-4b46-bed0-905ae1769bc6 Response Example HTTP/1.1 204 No Content
