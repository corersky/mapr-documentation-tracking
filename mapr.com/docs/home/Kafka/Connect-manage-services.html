Managing Kafka Connect Services Lists the commands you use to start, stop, or restart Kafka Connect Services Use the maprcli node services command with the -action parameter as follows: maprcli node services -name kafka-connect -action start -nodes maprcli node services -name kafka-connect -action stop -nodes maprcli node services -name kafka-connect -action restart -nodes For more information see [1]node services References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/node-services.html
