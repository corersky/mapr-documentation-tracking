POST /consumers/{string: group_name}/instances/{string: instance_id}/assignments Manually assigns a list of partitions to a consumer. Parameters CAPTION: Table 1. Parameters Parameter Description group_name (string) Name of the consumer group. instance (string) ID of the consumer instance. Syntax http://:8082/consumers//instances//assignments Request Example curl -X POST -H "Content-Type: application/vnd.kafka.v2+json" --data '{"partitions":[{"topic":"first","partition":0}]}' https://localhost:8082/consumers/grouptest/instances/user/assignments Response Example HTTP/1.1 204 No Content
