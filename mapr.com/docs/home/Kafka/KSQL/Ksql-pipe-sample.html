Pipe Code Sample Provides sample code for a Pipe example. The following is a Pipe code sample that moves an inputTopic to an outputTopic: ksql> CREATE STREAM stream3 (message varchar) WITH (kafka_topic='/sample-stream:inputTopic', value_format='DELIMITED'); ksql> CREATE STREAM stream4 (message varchar) WITH (kafka_topic='/sample-stream:streams-pipe-output1', value_format='DELIMITED') AS SELECT * FROM stream3;
