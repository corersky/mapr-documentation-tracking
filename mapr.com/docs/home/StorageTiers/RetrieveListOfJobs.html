Viewing the List of Running Jobs You can view the tiering jobs currently running for a volume using the CLI and REST API. For a tiered volume, at any given time, there can be only one running job. Run the following command or send a request of type GET to retrieve the list of currently running tiering operations for a volume: * [1]CLI * [2]REST maprcli volume tierjobstatus -name curl -k -X GET 'https://:8443/rest/volume/tierjobstatus?name=' -- user mapr:mapr For more information, see [3]volume tierjobstatus.
