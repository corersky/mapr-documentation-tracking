Starting, Stopping, and Restarting the MAST Gateway You can start, stop, and restart the MAST Gateway using the MapR Control System and the CLI. Starting, Stopping, and Restarting the MAST Gateway Using MCS See: * [1]Starting the Services on the Cluster Using the MapR Control System * [2]Stopping a Service on the Cluster Using the MapR Control System * [3]Restarting the Services on the Cluster Using the MapR Control System Starting, Stopping, and Restarting the MAST Gateway Using the CLI Run the following command to: * Start the MAST Gateway: maprcli node services -nodes -name mastgateway -action sta rt * Stop the MAST Gateway: maprcli node services -nodes -name mastgateway -action sto p * Restart the MAST Gateway: maprcli node services -nodes -name mastgateway -action res tart References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ClusterAdministration/nodes/Services-Start.html#mcs-clstr 2. file://localhost/root/docsync/tmp/mapr.com/docs/home/ClusterAdministration/nodes/Services-Stop.html#mcs-clstr 3. file://localhost/root/docsync/tmp/mapr.com/docs/home/ClusterAdministration/nodes/Services-Restart.html#mcs-clstr
