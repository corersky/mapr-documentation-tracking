Viewing the List of Storage Policies You can view the list of storage policies (rules) using the MapR Control System, CLI, and REST API. Viewing the List of Storage Policies Using the MapR Control System Log in to MCS and click Data > Volumes > Storage Policies. The list of storage policies displays. For each storage policy, the page displays the following: Column Name Column Description Policy Name The name of the policy. Detail The policy details. You can: * [1]Create a Policy * [2]Edit a Policy * [3]Remove a Policy Viewing the List of Storage Policies Using the CLI and REST API * [4]CLI * [5]REST Run the following command to retrieve the list of tiers: # maprcli tier rule list Send a request of type GET.
