Editing a Tier You cannot modify a warm tier. You can modify a cold tier (referred to as Remote Target in the MCS) using the MapR Control System, the CLI, and REST API. Modifying a Remote Target Using the MapR Control System 1. Log in to MCS and go to the Remote Target tab under Data > Volumes. 2. Click on the name of the remote target (cold tier) to display the Edit Remote Target window. 3. Make necessary changes to the CREDENTIALS. 4. Click Save Changes to save the changes. Modifying a Cold Tier Using the CLI and REST API * [1]CLI * [2]REST Run the following command to modify a cold tier: $maprcli tier modify -name -credential Send a request of type POST. For example: curl -k -X POST 'https://abc.sj.us:8443/rest/tier/modify?name=&creden tial_str=' --user mapr:mapr For more information, see [3]tier modify.
