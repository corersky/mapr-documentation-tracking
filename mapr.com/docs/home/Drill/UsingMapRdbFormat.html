MapR Database Format Plugin for Drill Drill supports access to MapR Database JSON and binary tables through the maprdb format plugin. When you install Drill, the maprdb format is automatically defined within the default dfs storage plugin configuration to make MapR Database a consumable data source for Drill. You can access the dfs storage plugin configuration from the Storage page in the [1]Drill Web UI. When you install Drill on MapR, Drill provides some options specific to the maprdb format plugin that you can change or configure. You can modify these options in the following places: * [2]dfs storage plugin configuration * [3]drill-override.conf file * [4]SET command For additional information about storage plugins, see [5]Plugin Configuration Basics.
