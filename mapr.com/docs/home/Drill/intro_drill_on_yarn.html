Drill-on-YARN You can install and run Drill under Warden or you can install and run Drill under YARN. [1]YARN (Yet Another Resource Negotiator) is a cluster management tool that automates the resource sharing process in a cluster. Note: The [2]MapR default security feature introduced in 6.0 is not supported with Drill-on-YARN.
