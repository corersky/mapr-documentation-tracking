Inserting an Apache Spark DataFrame into a MapR Database JSON Table Starting in the MEP 4.1.0 release, you can use the insertToMapRDB API to insert an Apache Spark DataFrame into a MapR Database JSON table in Python. The insertToMapRDB API throws an exception if a row with the same ID already exists.
