Prerequisites for Using Azure Data Lake Store To access Azure Data Lake Store (ADLS) from any application, you must have ADLS setup on the Azure portal. * Create an account on the [1]Azure portal. * Create an Azure Data Lake Store. For detailed information, see [2]Get started with Azure Data Lake Storage.
