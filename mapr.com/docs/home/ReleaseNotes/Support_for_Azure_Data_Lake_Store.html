Support for Azure Data Lake Store Starting with MapR release 6.1, you can use Azure Data Lake Store (ADLS) as an input or output for all applications. For general information about the features of ADLS, refer to the [1]Azure Data Lake Store documentation. You can also refer to the official [2]Apache documentation for configuring ADLS as a storage for a Hadoop cluster. References 1. https://docs.microsoft.com/en-in/azure/data-lake-store/ 2. https://hadoop.apache.org/docs/r2.8.0/hadoop-azure-datalake/index
