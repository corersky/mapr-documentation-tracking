Disabling Wire-level Security You can disable wire encryption for a volume using the MapR Control System, the CLI, and REST API. Disabling Wire-level Security for a Volume Using the MapR Control System 1. Log in to MCS and click Data > Volumes. 2. Click Create Volume to display the Create New Volume page or go to the [1]Edit Volume page. 3. Set the value for the Data on Wire Encryption property to No (to disable). See [2]Creating a Volume or [3]Modifying a Volume for more information. 4. Complete the steps to create or modify the volume. See [4]Creating a Volume or [5]Modifying a Volume for more information. Disabling Wire-Level Security for a Volume Using the CLI and REST API You can disable encryption of data on wire at the volume level. * [6]CLI * [7]REST Set the value for the wiresecurityenabled parameter to true when you: * Create the volume. For example: maprcli volume create -name -path -wiresecurityenabled false * Modify the volume. For example: maprcli volume modify -name -wiresecurityenabled false Send a request of type POST and set the value for the wiresecurityenabled parameter to true when you: * Create the volume. For example: curl -k -X POST 'https://abc.sj.us:8443/rest/volume/create?name=&path=< volMountPath>&wiresecurityenabled=false' --user mapr:mapr * Modify the volume. For example: curl -k -X POST 'https://abc.sj.us:8443/rest/volume/modify?name=&wirese curityenabled=false' --user mapr:mapr See [8]volume create and [9]volume modify for more information.
