Ports Used for Secure Services Lists the ports used by web interfaces for secure Hadoop 2 services The MapR software supports Hadoop 2 services built on the YARN framework. Hadoop 2 services use different ports from the HTTP and HTTPS protocols, listed in the following table: Service HTTP (non-secure) port HTTPS (secure) port NodeManager 8042 8044 ResourceManager 8088 8090 HistoryServer 19888 19890 Note: For a list of all ports used by MapR, see [1]Ports Used by MapR.
