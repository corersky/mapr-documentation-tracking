Verifying Selective Auditing of Operations After you set up the list of operations to include and/or exclude from auditing, you can retrieve and verify the list of included and/or excluded operations using the [1]maprcli volume info command. When you run the volume info command, the output will show the list of operations: * Excluded (disableddataauditoperations) from auditing. * Included (enableddataauditoperations) for auditing.
