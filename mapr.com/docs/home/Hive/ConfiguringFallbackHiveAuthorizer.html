Configuring Fallback Hive Authorizer The Fallback Hive Authorizer Factory provides access to Fallback Hive Authorizer via the hive.security.authorization.manager and hive.security.authorization.enabled properties. The default value for hive.security.authorization.enabled is true and hive.security.authorization.manager is org.apache.hadoop.hive.ql.security.authorization.plugin.fallback.Fallba ckHiveAuthorizerFactory on MapR SASL clusters. Configure the hive.users.in.admin.role property for administrator users. The default value for hive.users.in.admin.role is mapr on MapR SASL clusters.
