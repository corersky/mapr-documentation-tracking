Action Restrictions with Fallback Hive Authorizer After enabling Fallback Hive Authorizer, you can perform action restriction operations. * Disallow local file location in SQL statements for all except the administrator. * Allow set for selected white list parameters. * Disallow dfs commands for all except the administrator. * Disallow ADD JAR statements for all except the administrator. * Disallow COMPILE statements for all except the administrator. * Disallow TRANSFORM statements.
