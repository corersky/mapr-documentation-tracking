Understanding the UPDATE Statement Starting with MEP 6.0.0 (Hive 2.3), MEP 5.0.1 (Hive 2.1), MEP 4.1.2, and MEP 3.0.4, the UPDATE statement is supported with Hive MapR Database JSON tables. You can use the UPDATE statement to update primitive, complex, and complex nested data types in MapR Database JSON tables, using the Hive connector.
