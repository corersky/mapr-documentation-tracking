Upgrading Livy This section describes how to upgrade Livy without the MapR Installer. Run the following command to upgrade Livy using a package manager: * On RedHat/CentOS: yum update mapr-livy * On Ubuntu: apt-get install mapr-livy * On SUSE: zypper update mapr-livy Note: Livy is included in MEP repositories beginning with MEP 4.0.0. Before MEP 4.0.0, it was included as a package called mapr-hue-livy and released only as a part of Hue.
