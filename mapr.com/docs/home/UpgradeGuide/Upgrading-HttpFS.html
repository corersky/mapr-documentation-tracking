Upgrading HttpFS This section describes how to upgrade HttpFS without the MapR Installer. Run the following set of commands as root or with sudo: 1. Use one of the following methods to upgrade the HttpFS packages: + RedHat/CentOS yum remove mapr-httpfs rm -rf /opt/mapr/httpfs/ yum install mapr-httpfs + Ubuntu apt-get autoremove mapr-httpfs rm -rf /opt/mapr/httpfs/ apt-get install mapr-httpfs + SUSE zypper remove mapr-httpfs rm -rf /opt/mapr/httpfs/ zypper install mapr-httpfs
