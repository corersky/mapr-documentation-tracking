Upgrading Drill This section describes how to upgrade Drill without the MapR Installer. Important: Before upgrade Drill, complete the [1]pre-upgrade steps. Complete the following steps on the Drill server and client nodes as root or using sudo to upgrade Drill without the MapR Installer: 1. To eliminate cached packages and files, issue the following command : yum clean all 2. To upgrade Drill, issue the command appropriate for your system on each Drill node: + RedHat/CentOS yum update mapr-drill + Ubuntu sudo apt-get install mapr-drill + SUSE zypper update mapr-drill Note: SUSE is supported as of Drill 1.9.0-1703 and Drill 1.10.0-1703. 3. Complete the [2]post-upgrade steps for Drill.
