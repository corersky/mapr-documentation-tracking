Upgrading the MapR Data Access Gateway This section describes how to upgrade the MapR Data Access Gateway without the MapR Installer. Complete the following steps to upgrade the MapR Data Access Gateway without the MapR Installer. 1. Install the new package using the command that is appropriate for your distribution: RedHat/CentOS yum update mapr-data-access-gateway Ubuntu apt-get install mapr-data-access-gateway SUSE zypper update mapr-data-access-gateway 2. On the CLDB master node, use the manageSSLKeys.sh script to generate the p12 keystore file, which enables OpenSSL communication channel for the MapR gRPC service. For example: /opt/mapr/server/manageSSLKeys.sh convert -k -N /opt/mapr/conf/ss l_keystore /opt/mapr/conf/ssl_keystore.pem 3. Copy the /opt/mapr/conf/ssl_keystore.p12 file to all other nodes that contain the Data Access Gateway.
