Upgrading HBase REST Gateway Complete the following steps to upgrade the HBase REST Gateway: 1. Run the following command to upgrade the HBase REST package: On CentOS / Red Hat yum update mapr-hbase-rest mapr-hbase Ubuntu apt-get update mapr-hbase-rest mapr-hbase SUSE zypper update mapr-hbase-rest mapr-hbase 2. Run [1]configure.sh on the node where you upgrade the HBase Thrift package. References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/configure.sh.html#configure.sh
