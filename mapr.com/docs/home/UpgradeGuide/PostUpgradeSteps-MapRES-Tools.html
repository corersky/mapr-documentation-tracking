Post-Upgrade Steps for MapR Event Store For Apache Kafka Tools Complete the following steps after upgrading MapR Event Store For Apache Kafka Tools with or without the MapR Installer: Kafka REST Proxy for MapR Event Store For Apache Kafka To following post-upgrade steps are applicable when upgrading Kafka REST Proxy. 1. (Optional) Migrate your custom configuration settings into the new default files in the conf directory. For example: /opt/mapr/kafka-rest-/config 2. Run the configure.sh file. For example: /opt/mapr/server/configure.sh -R 3. Start the Kafka REST service maprcli node services -name kafka-rest -action start -nodes < list of Kafka REST service nodes >
