Pre-Upgrade Steps for Oozie Complete the following steps before you upgrade Oozie with or without the MapR Installer. 1. Stop any jobs or coordinators that are in a RUNNING or SUSPENDED state. 2. For upgrades without the MapR Installer: Stop the Oozie service. # maprcli node services -name oozie -action stop -nodes 3. (OPTIONAL) Back up Oozie configuration files and save them to a location outside the MapR installation. Configuration properties are located in /opt/mapr/oozie/oozie-/conf/. After upgrading, you can reapply changes to the updated Oozie installation using the backup. 4. For upgrades without the MapR Installer: Remove the old share libraries and examples from the following directories: maprfs:///oozie/share maprfs:///user/${user.name}/examples 5.
