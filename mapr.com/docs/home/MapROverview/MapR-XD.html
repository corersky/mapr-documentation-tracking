MapR XD Distributed File and Object Store MapR XD Cloud-Scale Data Store provides exabyte-scale data store for building intelligent applications with the MapR Converged Data Platform. MapR XD includes all the functionality you need to manage large amounts of conventional data. Why MapR XD? MapR XD can be installed on SSD- and HDD-based servers. It includes the MapR Filesystem for data storage, data management, and data protection, support for mounting and accessing the clusters using NFS and the FUSE-based POSIX (basic, platinum, or PACC) clients, and support for accessing and managing data using HDFS APIs. The cluster can be managed using the MapR Control System and monitored using MapR Monitoring (Spyglass initiative). MapR XD is the only Cloud-Scale Data store that enables you to build a fabric of exabyte scale. MapR XD supports trillions of files, 100s of 1000s of client nodes and can run on Edge Cluster, on-prem datacenters and the public cloud.
