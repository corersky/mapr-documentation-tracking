MapR Event Store For Apache Kafka MapR Event Store For Apache Kafka brings integrated publish and subscribe messaging to the MapR Converged Data Platform. Why MapR Event Store For Apache Kafka? MapR Event Store For Apache Kafka is built into the MapR platform. It requires no additional process to manage, leverages the same architecture as the rest of the platform, and requires minimal additional management.
