Apache Kafka Java APIs MapR Event Store For Apache Kafka supports these Apache Kafka Java APIs. Javadoc As of MapR 6.1, Apache Kafka 1.1 is supported. See the following APIs for detailed information: * [1]MapR Event Store For Apache Kafka Java API Library (6.1) * [2]Apache Kafka 1.1 APIs used with MapR Event Store For Apache Kafka (6.1) * [3]Admin APIs * [4]Consumer APIs * [5]Producer APIs * [6]Common APIs The following Admin APIs, org.apache.kafka.clients.admin package, are applicable to MapR support of Apache Kafka. These APIs are supported as of MapR 6.0.1. * [7]AdminClient Note: For MapR 6.1, the following Apache Kafka API is not supported: * deleteRecords() Note: The AdminClient API options (CreateTopicsOptions, DeleteTopicsOptions, DescribeTopicsOptions, ListTopicsOptions, CreatePartitionsOptions, and DescribeTopicsOptions), are ignored. All of the methods assume that a topic belongs to the default stream unless a stream path is specified in the topic name. If a default stream name is not specified and the topic path does not contain a stream name, the an exception is reported via the Result object. For example: * If the topic name is specified as topic1, then the API assumes the full topic path as /defaultStream:topic1. * If the topic name is specified as /defaultStream:topic1, then that will be the full topic path.
