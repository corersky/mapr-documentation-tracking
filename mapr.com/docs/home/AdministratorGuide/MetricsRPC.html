MapR RPC Metric Every 10 seconds, the collectd service uses a MapR plugin to gather the following MapR RPC metrics on each node in the cluster. Name Description mapr.rpc.bytes_recd The number of bytes received by the MapR-FS over RPC. mapr.rpc.bytes_sent The number of bytes sent by the MapR-FS over RPC. mapr.rpc.calls_recd The number of RPC calls received by the MapR-FS.
