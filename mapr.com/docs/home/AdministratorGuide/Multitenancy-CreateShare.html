Creating a Volume for a Tenant To create a volume for a tenant (when you have [1]Multitenancy on MapR Filesystem), do the following: 1. Log in to the cluster as administrator and ensure that the ticket for the tenant has already been generated and copied on to the tenant host. If necessary, follow steps in [2]Generating a Ticket for a Tenant to set up ticket for the tenant. 2. Create a volume for the tenant by running the following command: $ maprcli volume create -name -path -tenantuser Note: For more information, see [3]maprcli volume create command.
