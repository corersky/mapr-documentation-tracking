Setting Up Volume Topology to Restrict the CLDB Volume to Specific Nodes 1. Move all CLDB nodes to a CLDB-only topology (e. g. /cldbonly) using the MapR Control System or the following command: maprcli node move -serverids -topology /cldbonly 2. Restrict the CLDB volume to the CLDB-only topology using the MapR Control System or the following command: maprcli volume move -name mapr.cldb.internal -topology /cldbonly
