Setting up Policy To add a new policy for file upload: 1. Log in to the UI and click Add New File Migration Policy. [addPolicy.png] Alternatively, select Policy from the [gearMenu.png] dropdown menu. [addPolicy2.png] 2. Create policies in the Add Data Migration Policy page. To create a new policy, set the following: + Directory Path + Target Bucket + Purge Interval + Delete Empty Directories + Ignore Files Regex + X-Attributes See [1]FileMigrate.properties for more information. 3. Click Create New Policy. 4. Click Restart File Migration Service. References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/AdministratorGuide/PropertiesFile.html
