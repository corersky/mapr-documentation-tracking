Viewing Active Stream Alarms You can view stream alarms using MCS and the CLI. Viewing Active Stream Alarms in the MapR Control System Log in to MCS and: * Click Data > Streams to view all stream alarms in the Active Alarms pane. * Go to the Summary tab in the [1]stream information page to view the recent and active alarms for the selected stream in the Active Alarms pane. You can: * [2]View information related to the alarm. * [3]Dismiss an alarm. * [4]Mute an alarm. See [5]Alarms Reference for more information on the stream alarms. Retrieving Active Stream Alarms Using the CLI or REST API The basic command: maprcli alarm list -cluster -type volume For complete reference information, see [6]alarm list.
