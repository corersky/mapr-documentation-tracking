Managing the mapr-loopbacknfs Service To manually start or stop the service: service mapr-loopbacknfs [start|stop] To have the service start automatically when the OS starts up: chkconfig –add mapr-loopbacknfs chkconfig –list mapr-loopbacknfs # confirm settings To monitor the service: service mapr-loopbacknfs status showmount –e localhost The showmount command displays: Export list for /mapr 127.0.0.1 /mapr/ 127.0.0.1
