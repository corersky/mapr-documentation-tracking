MapR Monitoring Logs The fluentd component reads and parses the following MapR Monitoring component logs on each node in the cluster. CAPTION: Service Name Parsing Method Description collectd Multi-line The collectd component logs from $MAPR_HOME/collectd/collectd-*/var/log/collectd/collectd_daemon.log fluentd Multi-line The fluentd component logs from $MAPR_HOME/fluentd/fluentd-*/var/log/fluentd/fluentd.log, grafana Multi-line Grafana logs from $MAPR_HOME/grafana/grafana-*/var/log/grafana/grafana.log. kibana Multi-line Kibana logs from $MAPR_HOME/kibana/kibana-*/var/log/kibana/kibana.log.
