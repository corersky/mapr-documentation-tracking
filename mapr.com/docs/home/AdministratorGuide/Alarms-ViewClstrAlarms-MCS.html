Viewing Active Cluster Alarms You can view cluster alarms using MCS and the CLI. Viewing Active Cluster Alarms in the MapR Control System Log in to MCS and click: * Overview to view all the alarms on the cluster in the Active Alarms pane. To view only the cluster alarms, select Cluster Alarms from the dropdown menu in the Active Alarms pane. * [alarmSummary.png] to view all the alarms on the cluster in the All alarms pane. To view only the cluster alarms, select Cluster Alarms from the dropdown menu in the All alarms pane. You can: * [1]View alarm information * [2]Dismiss an alarm * [3]Mute an alarm Retrieving Cluster Alarms Using the CLI or REST API The basic command to retrieve the alarms for a cluster is: maprcli alarm list -cluster -type cluster For complete reference information, see [4]alarm list.
