Recovering Volumes from Mirrors Using the CLI 1. Use the [1]volume dump create command to create a full volume dump for each mirror volume you want to restore. Example: maprcli volume create -e statefile1 -dumpfile fulldump1 -name volume@cluster 2. Transport the volume dump to the rebuilt cluster. 3. For each volume on the mirror cluster, set up a corresponding volume on the rebuilt cluster. a. Restore the volume using the [2]volume dump restore command. Example: maprcli volume dump restore -name volume@cluster -dumpfile fulldump1 b. Copy the files to a standard (non-mirror) volume.
