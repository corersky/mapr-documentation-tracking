MapR I/O Metrics Every 10 seconds, the collectd service uses a MapR plugin to gather the following MapR I/O metrics on each node in the cluster. Name Description mapr.io.read_bytes The number of MB read from disk. mapr.io.reads The number of MapR Filesystem disk read operations. mapr.io.write_bytes The number of MB written to disk. mapr.io.writes The number of MapR Filesystem disk write operations.
