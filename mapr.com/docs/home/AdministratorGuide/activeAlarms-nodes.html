Viewing Active Node Alarms You can view the node alarms using MCS and the CLI. Viewing Active Node Alarms in the MapR Control System Log in to MCS and: * Click Nodes to view all the node alarms on the cluster in the Active Alarms pane. * Go to the [1]node information page to view alarms in the Alarms pane for the selected node. * Click [alarmSummary.png] (in the top navigation bar) to display the Alarm Summary page and select Node Alarms from the drop-down menu in the All alarms pane. * Click Overview and select Node Alarms from the drop-down menu in the Active Alarms pane to view all the node alarms on the cluster. You can: * [2]View alarm notes * [3]Mute an alarm * [4]Dismiss an alarm Retrieving Active Node Alarms Using the CLI or REST API The basic command to retrieve node alarms is: maprcli alarm list -cluster -type node For complete reference information, see [5]alarm list.
