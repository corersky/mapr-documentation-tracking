Running MapR Services as the Root User 1. Stop Warden. service mapr-warden stop 2. If ZooKeeper is installed on the node, stop it. service mapr-zookeeper stop 3. Run the script $INSTALL_DIR/server/config-mapr-user.sh -u root 4. If Zookeeper is installed, start it. service mapr-zookeeper start 5. Start Warden. service mapr-warden start
