Using the Command Line to View Logs for Completed Applications 1. Determine the application ID for the application that you want to view the logs for. For example, run the following command to list the applications: yarn application -list yarn application -list 2. Run the yarn logs command to view the logs for the application. For example, run the following command to view the log files for application application_1415822090718 yarn logs -applicationId application_1415822090718 yarn logs -applicationId application_1415822090718
