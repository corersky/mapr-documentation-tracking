6.1 Administration This section describes how to manage the nodes and services that make up a cluster. This section is for system administrators tasked with managing MapR clusters. Topics include how to manage data by using volumes, how to monitor the cluster for performance, how to manage users and groups, how to add and remove nodes from the cluster, and more. The focus of this section is managing the nodes and services that make up a cluster using the MapR Control System and the CLI or REST API.
