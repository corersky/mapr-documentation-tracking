Memory Metrics Every 10 seconds, the collectd service uses the memory and swap plugins to gather the following memory metrics on each node in the cluster. CAPTION: Name Description Additional Tag(s) memory.memory The amount of physical memory in bytes. * memory_type: Display values for a specified memory type. Memory type values: free, used, buffered, etc. swap.swap The amount of swap space in bytes. * swap_type: Display values for a specified swap type. Swap type values: used and free. swap.swap_io The amount of swap I/O in bytes.
