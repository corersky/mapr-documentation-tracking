Checking the Status of Oozie Once Oozie is installed, you can check the status using the command line or the Oozie web console. The [1] noted in the URLs depend on whether your cluster is secure. Check Status from Command Line Use the oozie admin command: /opt/mapr/oozie/oozie-/bin/oozie admin -oozie http://:/oozie -status The following output indicates normal operation: System mode: NORMAL Check Status from the Oozie Web Console Point your browser to http://:/oozie Note: * For non-secure clusters: Oozie url: http://:11000/oozie * For secure clusters: Oozie url: https://:11443/oozie References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/AdvancedInstallation/InstallOozie.html#InstallOozie__step_djg_dp4_qbb
