Configuring an Oracle Schema You must create schemas for Oracle databases manually. Create an Oozie SQL-with-schema file to reveal the schema: bin/ooziedb.sh create -sqlfile oozie.sql -runValidate DB Connection. See [1]Apache Oozie documentation for more information. References 1. https://oozie.apache.org/docs/5.0.0/AG_Install.html#Database_Configuration
