Oozie 4.3.0 API Changes Oozie 4.3.0 includes the following changes in the Web Services API. Full documentation is available at [1]http://oozie.apache.org/docs/4.3.0/WebServicesAPI.html API Description Validate a local file This endpoint is to validate a workflow, coordinator, bundle XML file. Validate a file in MapR Filesystem You can validate a workflow, coordinator, bundle XML file in MapR Filesystem. The XML file must already exist in MapR Filesystem.
