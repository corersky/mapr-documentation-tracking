What's New in MEP 4.1.0 Provides a summary of the new functionality in MEP 4.1.0. MEP 4.1.0 includes support for MapR 6.0 and these new features: New Features and Additions Support for Java and Python APIs for MapR Database OJAI Connector for Apache Spark. For more information, see [1]Understanding the MapR Database OJAI Connector for Spark. Support for saving an Apache Spark Dataset to a MapR Database table. For more information, see [2]Saving an Apache Spark Dataset to a MapR Database JSON Table. Support for insertToMapRDB API to insert an RDD, Dataframe, or Dataset into a MapR Database table. For more information, see [3]Saving Data to a MapR Database JSON Table. New Drill Version MEP 4.1.0 includes support for Drill 1.12. For more information, see [4]Drill 1.12.0-1801 Release Notes.
