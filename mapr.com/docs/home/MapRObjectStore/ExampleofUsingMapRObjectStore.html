Sample Usage of MapR Object Store The example in this section creates a bucket and uploads a file, moves an object from one bucket to another, deletes an object from a bucket, deletes the bucket, lists all files in the bucket, and shows all files on the file system. # Create a bucket and upload a file aws s3 mb s3://mybucket --endpoint-url https://:9000 --no-verify-ssl echo "HelloS3World" > /tmp/helloS3world.txt aws s3 cp /tmp/helloS3world.txt s3://mybucket/helloS3world.txt --endpoint-url ht tps://:9000 --no-verify-ssl # Move an object from one bucket to another aws s3 mv s3://firstbucket/helloS3world.txt s3://mybucket/helloS3world.txt --end point-url https://:9000 --no-verify-ssl # Delete an object from a bucket aws s3 rm s3://mybucket/helloS3world.txt --endpoint-url https://:900 0 --no-verify-ssl # Delete the bucket aws s3 rb s3://mybucket --endpoint-url https://:9000 --no-verify-ssl # List all files in the bucket aws s3 ls s3://mybucket/ --endpoint-url https://:9000 --no-verify-ss l # Show all files on the filesystem ls -al /mapr/demo.mapr.com/apps/s3/$(hostname)/mybucket/
