Troubleshooting HttpFS To debug authentication issues, follow these steps: 1. Edit the log4j properties file located at /opt/mapr/httpfs/httpfs-1.0/etc/hadoop/httpfs-log4j.properties and insert the following lines to activate debug capabilities: log4j.logger.org.apache.hadoop.fs.http.server=DEBUG, httpfs log4j.logger.org.apache.hadoop.lib=DEBUG, httpfs log4j.logger.org.apache.hadoop.security.authentication.server=DEBUG, httpfs 2. Search the logs located at /opt/mapr/httpfs/httpfs-1.0/logs for the words ERROR or Exception.
