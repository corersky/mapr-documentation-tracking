Changing Your Password Once you log in, you can change your password by following these steps: 1. Click on Username at the top right of the menu bar and select Edit Profile from the drop-down menu. In this example, the username is mapr. [28443169.png] The Hue Users dialog box opens. 2. Enter a new password in the Password field. [28443168.png] 3. Retype the password in the Password confirmation field. 4. Click the Update user button.
