Step 2: Configure Hue for MapR Database To configure Hue for MapR Database, edit the hbase section of the hue.ini file, which looks like this: [hbase] # Comma-separated list of HBase Thrift servers for # clusters in the format of '(name|host:port)'. ## hbase_clusters=(Cluster|localhost:9090) # Hard limit of rows or columns per row fetched before truncating. ## truncate_limit = 500 In this file, make the following changes: 1. Uncomment the ## hbase_clusters=(Cluster|localhost:9090) statement and provide the list of HBase Thrift servers. hbase_clusters=(|:9090),(|:909 0)[,...] 2. Uncomment the truncate_limit statement and change the value if necessary.
