volume balancinginfo Fetch currently running or scheduled balancer information for one or more volumes. Note: For best results, use the -json option when running the command. Syntax CLI maprcli volume balancinginfo [ -cluster cluster_name ] [ -name ] REST API N/A Parameters CAPTION: Parameter Description cluster The name of the cluster to run the command on. name The name of the volume. Output The command returns the following fields: CAPTION: Field Description spId The ID of the storage pool. capacity The capacity/size of the storage pool in MB. usedSize The size of the storage pool that is consumed. desiredSize The total size of the containers of a volume that should be allocated on this storage pool. isUnderweight Value is true if the storage pool contains less than 50% of the desiredSize for this volume. isOverweight Value is true if the storage pool contains more than 1.5 times the desiredSize for this volume.
