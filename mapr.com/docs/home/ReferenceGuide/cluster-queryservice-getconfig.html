cluster queryservice getconfig Retrieves the configuration of the OJAI Distributed Query Service. Permissions Required Only the user that enabled the OJAI Distributed Query Service can run this command. If the service was enabled during installation, user 'mapr' must run the command. Syntax CLI maprcli cluster queryservice getconfig -cluster < cluster-name > REST Not available Parameters Parameter Description cluster (Optional) Name of the cluster that is using secondary indexes to query MapR-DB JSON table Example maprcli cluster queryservice getconfig -cluster my.cluster.com
