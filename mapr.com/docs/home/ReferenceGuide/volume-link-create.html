volume link create Creates a link to a volume. Permissions required: fc or m on the volume. Syntax CLI maprcli volume link create [ -cluster ] -path -type -volume REST Request Type POST Request URL http[s]://:/rest/volume/link/create? Parameters CAPTION: Parameter Description path The -path parameter specifies the link path: /link Example: /home/abc/.rw type The volume type: writeable or mirror. volume The volume name. cluster The cluster name. Examples Create a link to v1 at the path v1. mirror: CLI maprcli volume link create -volume v1 -type mirror -path /v1.mirror REST https://abc.sj.us:8443/rest/volume/link/create?path=/v1.mirror&type=mirror&volum e=v1
