volume move Moves the specified volume or mirror to a different topology. Permissions required: m or fc on the volume. Syntax CLI maprcli volume move [ -cluster ] -name -topology REST Request Type POST Request URL http[s]://:/rest/volume/move? Parameters Parameter Description cluster The name of the cluster on which to run the command. name The name of the volume to move. topology The new rack path to the volume. To move the volume to another topology, you must have the [1]Converged Enterprise Edition installed on your system. Without the Converged Enterprise Edition, when you run the maprcli volume move command to move the volume to another topology, the following error message is returned: ERROR (10010) - Volume Move: No license for requested operation Examples * [2]CLI * [3]REST maprcli volume move -name testVolume -topology /newPath curl -k -X 'https://abc.sj.us:8443/rest/volume/move?name=testVolume&topology=%2F newPath' --user mapr:mapr References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/UpgradeGuide/License-upgrade.html#license-upgrade 2. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/volume-move.html#div1entry1 3. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/volume-move.html#div1entry2
