file recall Initiates recall of a file from a storage tier using a MAST Gateway. Permissions Required The user running the command must have (mode bit or [1]ACE) permissions to write to the file. Syntax CLI maprcli file recall -name REST Request Type GET Request URL http[s]:///rest/file/recall? Parameters CAPTION: Parameter Description name The name (including the path) of the file to recall.
