MapR User Mismatch UI Column MapR User Mismatch Alarm Logged As NODE_ALARM_MAPRUSER_MISMATCH Meaning The cluster nodes are not all set up to run MapR services as the same user (for example, some nodes are running MapR as root while others are running as mapr_user. Resolution For the nodes on which the User Mismatch alarm is raised, follow the steps in [1]Changing the User for MapR Services from the Command-Line.
