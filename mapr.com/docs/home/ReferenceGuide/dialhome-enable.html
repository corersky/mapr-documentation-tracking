dialhome enable Enables Dial Home on the cluster. Permissions required: fc or a Syntax CLI maprcli dialhome enable -enable 0|1 REST Request Type POST Request URL http[s]://:/rest/dialhome/enable Parameters CAPTION: Parameter Description enable Specifies whether to enable or disable Dial Home: * 0 - Disable * 1 - Enable Output A success or failure message.
