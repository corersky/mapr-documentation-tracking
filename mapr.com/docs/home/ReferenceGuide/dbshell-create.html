dbshell create Describes how to use MapR Database shell to create a JSON table. Description To create JSON tables, run the create command. When you create a JSON table using mapr dbshell, the default column family is created automatically. There are no commands for creating additional column families. To create column families, exit the shell and use the [1]maprcli table cf create command. Note: If you are using a 5.2.x dbshell client to connect to a 6.0 (or later) server, you must set the insertionorder table option to false before creating your table. The insertionorder option is not support as of 6.0. See the following example: tableoptions --insertionorder false Run tableoptions in dbshell to see the current setting of insertionorder.
