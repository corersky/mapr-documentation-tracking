tier rule remove Removes the rule for offloading data. Note: You cannot remove a rule that is currently associated with a volume. Syntax CLI $ maprcli tier rule remove -name [ -cluster ] REST Request Type POST Request URL http[s]:///rest/tier/rule/remove? Parameters Parameter Description cluster The name of the cluster on which to run the command. name The name of the rule to remove.
