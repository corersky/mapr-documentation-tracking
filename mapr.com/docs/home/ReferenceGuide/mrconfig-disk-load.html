mrconfig disk load After initializing a disk with the [1]mrconfig disk init command, load the disk into memory with the mrconfig disk load command. See [2]mrconfig for instructions about running mrconfig commands. Syntax /opt/mapr/server/mrconfig disk load Parameters CAPTION: Parameter Description path The device path of the disk; example /dev/sdc Examples Load a disk on a local node /opt/mapr/server/mrconfig disk load /dev/sdc References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/mrconfig-disk-init.html 2. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/mrconfig.html
