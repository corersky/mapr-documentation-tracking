schedule create Creates a schedule. Permissions required: fc or a. A schedule can be associated with a volume to automate mirror syncing and snapshot creation. See [1]volume create and [2]volume modify. Syntax CLI maprcli schedule create [ -cluster ] -schedule REST Request Type POST Request URL http[s]://:/rest/schedule/create? Parameters Parameter Description cluster The cluster on which to run the command. schedule A JSON object describing the schedule. See [3]Schedule Objects for more information.
