mastgateway tierget Test retrieving an object from the storage tier. Syntax mrconfig mastgateway tierget [ ] Parameters CAPTION: Parameter Description isSecure Specifies whether to use HTTPs or HTTP protocol. Value can be one of the following: * true - for HTTPs protocol * false - for HTTP protocol objectID The ID of the object to get frp, the storage tier. The ID must be the same ID (in string format) specified when offloading the object (using tierput command). objectSize The size of the object to get from the storage tier. The default value is 64KB. tierName The name of the storage tier. If necessary, run the [1]tier list command to retrieve the names of the tiers. Example Retrieve the object named sampleamazonobj of size 20971520 on the tier named amazonTier: # /opt/mapr/server/mrconfig mastgateway tierget amazonTier sampleamazonobj true 20971520 time take for the operation: 8.748000 seconds tierget successful References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/tier-list.html
