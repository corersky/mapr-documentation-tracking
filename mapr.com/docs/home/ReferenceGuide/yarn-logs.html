yarn logs Dumps the YARN container logs. Syntax yarn logs -applicationID [-appowner ] [-containerId ] [-nodeAddress ] Parameters CAPTION: Parameter Description -applicationID Specifies an application Id. -appowner Specifies the application owner. This defaults to the current user if this option is not specified. -containerID Specifies the container ID. This is required when -nodeAddress is specified. -nodeAddress Specifies the node address in the following format: nodename:port This is required when -containerID is specified.
