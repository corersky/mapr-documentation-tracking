setloglevel fileserver Sets the log level on the FileServer service. Permissions required: fc or a. Syntax CLI maprcli setloglevel fileserver -classname -loglevel DEBUG|ERROR|FATAL|INFO|TRACE|WARN -node -port REST Request Type POST Request URL http[s]://:/rest/setloglevel/fileserver? Parameters Parameter Description classname The name of the class for which to set the log level. The classname is listed under the maprcli trace info command. Contact MapR Support for this parameter. loglevel The log level to set. Default: INFO * DEBUG * ERROR * FATAL * INFO * TRACE * WARN node The node on which to set the log level. port The MapR-FS port. Default: 5660 Examples * [1]CLI * [2]REST maprcli setloglevel fileserver -classname FileServer -loglevel debug -node centos26.lab https://abc.sj.us:8443/rest/setloglevel/fileserver?classname=FileServer&loglevel =debug&node=centos26.lab References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/setloglevel-fileserver.html#div1entry1 2. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/setloglevel-fileserver.html#div1entry2
