virtualip move Reassigns a virtual IP or a range of virtual IP addresses to a specified Media Access Control (MAC) address. Syntax CLI maprcli virtualip move [ -cluster ] -virtualip [ -virtualipend -tomac REST Request Type POST Request URL http[s]://:/rest/virtualip/move? Parameters Parameter Description cluster The name of the cluster where the virtual IP addresses are being moved. tomac The MAC address that the virtual IP addresses are being assigned. virtualip A virtual IP address. If you provide a value for -virtualipend, this virtual IP address defines the beginning of the range. virtualipend A virtual IP address that defines the end of a virtual IP address range. Examples Move a range of three virtual IP addresses to a MAC address for the cluster my.cluster.com: CLI maprcli virtualip move -cluster my.cluster.com -virtualip 192.168.0.8 -virtualip end 192.168.0.10 -tomac 00:FE:ED:CA:FE:99 REST curl -k -X POST 'https://abc.sj.us:8443/rest/virtualip/move?cluster=my.cluster.c om&virtualip=192.168.0.8&virtualipend=192.168.0.10&tomac=00%3AFE%3AED%3ACA%3AFE% 3A99' --user mapr:mapr
