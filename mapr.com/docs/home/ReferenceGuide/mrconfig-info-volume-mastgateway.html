mrconfig info volume mastgateway The mrconfig info volume mastgateway command displays the volume assignment information. This command must be run on a CLDB node. See [1]mrconfig for instructions about running mrconfig commands. Syntax /opt/mapr/server/mrconfig info volume mastgateway [] Parameters CAPTION: Parameter Description volName The name of the volume for which to retrieve the volume assignment information.
