cluster queryservice Describes the commands to enable/disable and view the settings for the OJAI Distributed Query Service. Enable the [1]OJAI Distributed Query Service if you want the following functionality when querying MapR-DB JSON tables: * Advanced secondary index selection * Sorts of large data sets * Parallel query execution Permissions Required If you enable the OJAI Distributed Query Service during installation, then you must be user 'mapr' to run these commands. If you disable the service, and later re-enable it, then the command needs to be run by the user that re-enabled the service.
