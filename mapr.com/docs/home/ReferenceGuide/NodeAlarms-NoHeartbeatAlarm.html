No Heartbeat Alarm Describes the NODE_ALARM_NO_HEARTBEAT alarm. UI Column No Heartbeat Alarm Logged As NODE_ALARM_NO_HEARTBEAT Meaning Node is not undergoing maintenance, and no heartbeat detected for over 5 minutes. Resolution Check the status of the node manually. Configuration Configurable at cluster level. See [1]Configuring the Alarm Threshold Using the CLI for more information. This alarm is raised when a node is down for more than 5 minutes. By default, this alarm is not raised if an edge node is down.
