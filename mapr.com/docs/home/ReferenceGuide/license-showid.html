license showid Displays the cluster ID for use when creating a new license. Permissions required: login. Syntax CLI maprcli license showid [ -cluster ] REST Request Type GET Request URL http[s]://:/rest/license/showid[?] Parameters Parameter Description cluster The cluster on which to run the command.
