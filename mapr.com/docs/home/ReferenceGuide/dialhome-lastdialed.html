dialhome lastdialed Displays the date of the last successful Dial Home call. Permissions required: fc or a. Syntax CLI maprcli dialhome lastdialed REST Request Type GET Request URL http[s]://:/rest/dialhome/lastdialed Output The date of the last successful Dial Home call. Sample output $ maprcli dialhome lastdialed date 1322438400000 Examples Show the date of the most recent Dial Home: CLI maprcli dialhome lastdialed REST https://abc.sj.us:8443/rest/dialhome/lastdialed
