file offload Initiates offload of a file using a MAST Gateway. Permissions Required The user running the command must have (mode bit or [1]ACE) permissions to write to the file. Syntax CLI maprcli file offload -name REST Request Type POST Request URL http[s]:///rest/file/offload? Parameters CAPTION: Parameter Description name The name (including the path) of the file to offload.
