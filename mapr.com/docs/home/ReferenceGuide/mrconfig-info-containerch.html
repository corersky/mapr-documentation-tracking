mrconfig info containerchain The mrconfig info containerchain command displays the containerchain for a given container. Example: $ /opt/mapr/server/mrconfig info containerchain 2050 Container 2050 prev 256000049 next 0. Container 256000049 prev 0 next 2050. See [1]mrconfig for instructions on running mrconfig commands. Syntax mrconfig [-h ] [-p ] info containerchain Parameters Parameter Description -h host IP address; default 127.0.0.1 -p The MapR Filesystem port; default 5660 cid The container identifier Tip: Use the [2]mrconfig info dumpcontainers command to find the container identifiers on a node. Examples Find the containerchain for a container with a cid of 2049 on a local node /opt/mapr/server/mrconfig info containerchain 2049 Find the containerchain for a container with a cid of 2049 on a remote node with an IP address of xx.xx.xx.xx /opt/mapr/server/mrconfig -h xx.xx.xx.xx info containerchain 2049 References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/mrconfig.html 2. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/mrconfig-info-dumpcontainers.html
