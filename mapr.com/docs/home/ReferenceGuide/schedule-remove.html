schedule remove Removes a schedule. A schedule can only be removed if it is not associated with any volumes. See [1]volume modify. Syntax CLI maprcli schedule remove [ -cluster ] -id REST Request Type POST Request URL http[s]://:/rest/schedule/remove? Parameters Parameter Description cluster The cluster on which to run the command. id The ID of the schedule to remove.
