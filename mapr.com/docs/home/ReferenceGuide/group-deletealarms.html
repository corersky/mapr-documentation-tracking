alarm group deletealarms Delete alarms in an alarm group. Syntax CLI maprcli alarm group deletealarms -groupname -alarms REST API N/A Parameters Parameter Description groupname The name of the alarm group to remove alarms from. alarms The comma-separated list of alarms to remove from the group. Examples Delete NODE_ALARM_HB_PROCESSING_SLOW and CLUSTER_ALARM_CLUSTER_ALMOST_FULL alarms in the cldb.alarm.group.info group: maprcli alarm group deleteAlarms -groupname cldb.alarm.group.info -alarms NODE_ALARM_HB_PROCESSING_SLOW,CLUSTER_ALARM_CLUSTER_ALMOST_FULL
