node listcldbs Returns the hostnames of the nodes in the cluster that are running the CLDB service. Syntax CLI maprcli node listcldbs [ -cluster ] [ -cldb ] REST Request Type GET Request URL http[s]://:/rest/node/listcldbs[?] Parameters CAPTION: Parameter Description cluster The name of the cluster for which to return the list of CLDB node hostnames. cldb The hostname or IP address and port number of a CLDB node.
