volume upgradeformat Upgrades and old-type volume to a new-type volume, which can in turn be used as a promotable mirror volume. Permissions required: m or fc on the volume. Syntax CLI maprcli volume upgradeformat [ -cluster ] -name REST Request Type POST Request URL http[s]://:/rest/volume/upgradeformat? Parameters CAPTION: Parameter Description cluster The cluster on which to run the command. name The volume name.
