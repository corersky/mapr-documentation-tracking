cluster mapreduce set Sets the cluster-wide MapReduce mode. Warning: This command is deprecated alongside MapReduce v1. Syntax CLI maprcli cluster mapreduce set -mode yarn REST http[s]://:/rest/cluster/mapreduce/set? Parameters CAPTION: Parameter Description mode The MapReduce mode of the cluster. Enter yarn to use Resource Manager and Node Manager to run MapReduce jobs or applications. Examples Sets the MapReduce mode for the cluster to classic. CLI maprcli cluster mapreduce set -mode yarn REST https://r1n1.sj.us:8443/rest/cluster/mapreduce/set?mode=yarn
