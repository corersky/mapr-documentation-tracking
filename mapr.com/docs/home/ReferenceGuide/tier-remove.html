tier remove Removes a tier. Note: You cannot remove a tier currently associated with a volume. Syntax CLI $ maprcli tier remove -name [ -cluster ] REST Request Type POST Request URL http[s]:///rest/tier/remove? Parameters CAPTION: Parameter Description cluster The name of the cluster on which to run the command. name The name of the tier to remove.
