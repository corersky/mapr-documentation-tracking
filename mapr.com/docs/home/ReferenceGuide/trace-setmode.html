trace setmode Sets the trace mode. There are two modes: * Default * Continuous In default mode, all trace messages are saved in a memory buffer. If there is an error, the buffer is dumped to stdout. In continuous mode, every allowed trace message is dumped to stdout in real time. Syntax CLI maprcli trace setmode [ -host ] -mode default|continuous [ -port ] REST None. Parameters CAPTION: Parameter Description host The IP address of the host on which to set the trace mode mode The trace mode. port The port to use. Examples Set the trace mode to continuous: CLI maprcli trace setmode -mode continuous
