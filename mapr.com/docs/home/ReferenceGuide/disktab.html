disktab On each node, the file /opt/mapr/conf/disktab lists all of the physical drives and partitions that have been added to MapR Filesystem. The disktab file is created by [1]disksetup and automatically updated when disks are added or removed (either using the MapR Control System, or with the [2]disk add and [3]disk remove commands).
