mrconfig cntr disablethrottle The mrconfig cntr disablethrottle command allows you to disable throttling for resync of a container (specified by ID). Run this command on the node that is the source for the resync. Note: By default, throttling will be disabled if resync is not complete after 30 minutes or when there is only one replica container. Syntax /opt/mapr/server/mrconfig cntr disablethrottle Parameters CAPTION: Parameter Description cid The ID of the container. Example Command # /opt/mapr/server/mrconfig cntr disablethrottle 2049 Output ---------------------- |From Instance 5660::| ---------------------- Changing throttling on container 2049 throttle flag disable
