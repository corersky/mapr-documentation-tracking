daemon.conf The file /opt/mapr/conf/daemon.conf specifies the user and group under which MapR services run, and whether all MapR services run as the specified user/group, or only ZooKeeper and FileServer. The configuration parameters operate as follows: * If mapr.daemon.user and mapr.daemon.group are set, the ZooKeeper and FileServer run as the specified user/group. Otherwise, they run as root. * If mapr.daemon.runuser.warden=1, all services started by the warden run as the specified user. Otherwise, they run as root. Sample daemon.conf file mapr.daemon.user=mapr mapr.daemon.group=mapr mapr.daemon.runuser.warden=1
