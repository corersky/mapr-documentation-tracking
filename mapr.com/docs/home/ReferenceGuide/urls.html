urls Displays the status page URL for the specified service. Syntax CLI maprcli urls [ -cluster ] -name [ -zkconnect ] REST Request Type GET Request URL http[s]://:/rest/urls/statuspage? Note: If you submit the request without statuspage in the request URL, the return contains a 404 error because the API requires a subcommand (empty or otherwise) to be present even though the subcommand is ignored.
