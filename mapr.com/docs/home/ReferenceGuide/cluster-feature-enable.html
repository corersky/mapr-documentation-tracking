cluster feature enable Allows features to be enabled. Used after upgrading. Note: Run [1]cluster feature list command to retrieve the list of features that can be enabled. Syntax CLI maprcli cluster feature enable [ -name ] [ -force ] [ -all ] REST N/A Parameters CAPTION: Parameter Description name Name of the feature. force Forces the enabled and disabled dependency features. No value is taken. all Enables all of the features. No value is taken. Note: Once a feature is enabled, it can not be disabled. Examples To enable all features, use the cluster feature enable -all command. maprcli cluster feature enable -all To enable a specific feature, use the cluster feature enable -name command. For example: maprcli cluster feature enable -name mfs.feature.audit.support References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ReferenceGuide/cluster-feature-list.html
