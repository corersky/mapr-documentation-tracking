cluster feature list Allows features to be listed. Used after upgrading. Syntax CLI maprcli cluster feature list [ -name ] [ -enabled ] [ -disabled ] REST N/A Parameters CAPTION: Parameter Description name Lists the named feature. Displays whether the feature is enabled or disabled along with the feature name. enabled Lists only the enabled features. No value is taken. disabled Lists only the disabled features. No value is taken. Tip: The three dots in the output indicate multiple entries. Use -json to format the output. Examples Lists the disabled features. maprcli cluster feature list -disabled dependency name description enabled mfs.feature.audit.support false Lists all the features. maprcli cluster feature list
