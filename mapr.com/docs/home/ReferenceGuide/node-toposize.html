node toposize Lists disk space utilization for each topology. Syntax CLI maprcli node toposize [ -cluster ] REST Request Type GET Request URL http[s]://:/rest/node/toposize[?] Parameters Parameter Description cluster The cluster on which to run the command. Output Field Description AvailableSpace The amount of available disk space (in GB). path The cluster rack path to the node (or topology of the node). TotalSpace The amount of total disk space (in GB). UsedSpace The amount of utilized disk space (in GB).
