volume unmount Unmounts one or more mounted volumes. Permissions required: fc or m on the volume. Syntax CLI maprcli volume unmount [ -cluster ] [ -force 0|1 ] -name REST Request Type POST Request URL http[s]://:/rest/volume/unmount? Parameters Parameter Description cluster The cluster on which to run the command. force Specifies whether (1) or not (0) to force the volume to unmount. name The name of the volume to unmount.
