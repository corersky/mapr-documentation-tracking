volume mount Mounts one or more specified volumes. Permissions required: fc or m on the volume. Syntax CLI maprcli volume mount [ -cluster ] -name -path [ -createparent 0|1 ] REST Request Type POST Request URL http[s]://:/rest/volume/mount? Parameters CAPTION: Parameter Description cluster The cluster on which to run the command. name The name of the volume to mount. path The path at which to mount the volume. The path must be relative to / and cannot be in the form of a global namespace path (for example, /mapr//). createparent Specifies whether or not to create a parent volume: * 0 = Do not create a parent volume. * 1 = Create a parent volume.
