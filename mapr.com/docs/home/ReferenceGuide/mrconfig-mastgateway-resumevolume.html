mastgateway resumevolume Resume tiering activities and allow reads and writes on the volume data. Syntax mrconfig mastgateway resumevolume Parameters Parameter Description volname The name of the volume. Examples # /opt/mapr/server/mrconfig mastgateway resumevolume volTSECNEW9_3 2018-08-06 02:47:14,8585 ERROR Global mrconfig.cc:2120 ResumeVolume succeed for volume : volTSECNEW9_3
