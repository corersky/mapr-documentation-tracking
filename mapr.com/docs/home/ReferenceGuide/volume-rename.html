volume rename Renames the specified volume or mirror. Permissions required: fc or d on the volume. Note: If you rename a volume, you must [1]unmount and [2]re-mount the volume to allow applications and/or users to continue accessing the volume. Syntax CLI maprcli volume rename [ -cluster ] -name -newname REST Request Type POST Request URL http[s]://:/rest/volume/rename? Parameters CAPTION: Parameter Description cluster The cluster on which to run the command. name The volume name. newname The new volume name. For tiering-enabled volumes, volume name cannot exceed ninety-eight characters.
