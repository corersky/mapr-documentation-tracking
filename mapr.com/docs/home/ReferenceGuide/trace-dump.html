trace dump Dumps the contents of the trace buffer into the MapR file system log. Syntax CLI maprcli trace dump [ -host ] [ -port ] REST None. Parameters CAPTION: Parameter Description host The IP address of the node from which to dump the trace buffer. Default: localhost port The port to use when dumping the trace buffer. Default: 5660 Examples Dump the trace buffer to the MapR file system log: CLI maprcli trace dump
