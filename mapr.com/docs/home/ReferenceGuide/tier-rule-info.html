tier rule info Retrieves information on a rule (specified by name). Syntax CLI maprcli tier rule info -name [ -output verbose ] [ -cluster ] REST Request Type GET Request URL http[s]:///rest/tier/rule/info? Parameters CAPTION: Parameter Description cluster The name of the cluster on which to run the command. name The name of the rule. output The type of output. The default value is verbose. Output The command returns the following: CAPTION: expression The rules defined using a combination of expressions. inuse Whether (true) or not (false) the rule is associated with a volume. rulename The name of the rule. ruleid The ID of the rule.
