Sqoop1 Sqoop transfers data between MapR Filesystem and relational databases. You can use Sqoop to transfer data from a relational database management system (RDBMS) such as MySQL or Oracle into MapR Filesystem and use MapReduce on the transferred data. Sqoop can export this transformed data back into an RDBMS.
