Configuring NFSv4 Server Without Kerberos To start using NFSv4 server without Kerberos, do the following: 1. Start the NFSv4 server. For more information, see [1]Starting, Stopping, and Restarting MapR NFSv4. 2. Verify that the list-exports command runs successfuly. For example: # maprcli nfs4mgmt list-exports Export Id Path 30 /mapr 0 / 3. Mount the cluster by running the mount command. For example: mount -t nfs4 :/ References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/ClusterAdministration/services/StartStopRestartNFSv4.html
