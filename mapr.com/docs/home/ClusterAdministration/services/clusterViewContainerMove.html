Viewing the List of Active Container Moves Log in to MCS and go to the [1]service information page for CLDB. The Active Container Moves section displays the following: CAPTION: Column Name Column Description Container ID The ID of the container being moved. Size The size of the container being moved. From Location The ID of From Storage Pool The ID of the source storage pool from where the container is being moved. To Location The ID of the To Storage Pool The ID of the destination storage pool for the move.
