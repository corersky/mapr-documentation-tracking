Viewing the List of Disks You can view the disks on a node using MCS and the CLI. Viewing the List of Disks Using the MapR Control System for MapR v6.0.1 To view both the system and file system disks: 1. Log in to MCS and go to the [1]node information page. 2. Go to the Summary tab. On this page, the Disks and System Disks panes display the following for each disk: Column Name Column Description Status The status of the disk. Value can be one of the following: + [diskStatus-ActiveGood.png] — indicates disk is active or good. + [diskStatus-Standby.png] — indicates disk is on standby. + [diskStatus-Offline.png] — indicates disk is offline. Device The disk partition(s). Mnt Indicates whether ( [checkMark.png] ) or not the disk is mounted. MapR-FS (Displayed in the Disks pane only) The disks available for MapR-FS. A [checkMark.png] indicates that the disk was added to MapR-FS. File System (Displayed in the System Disks pane only) The disks for system use. Allocated The amount of space allocated, in gigabytes. Used The percentage of disk space used. Model# The disk model number. Firmware Version The disk firmware version. Storage Pool The ID of the storage pool associated with the disk. Select the checkbox beside a disk in the Disks pane to: + [2]Add Disk(s) to MapR-FS + [3]Remove Disk(s) from MapR-FS Note: Disks in the System Disks pane cannot be selected. Viewing the List of Disks Using the MapR Control System for MapR v6.0.0 To view both the system and file system disks: 1. Log in to MCS and go to the [4]node information page. 2. Go to the Summary tab. On this page, for each disk on the node, the Disks pane display the following: Column Name Column Description Status The status of the disk. Value can be one of the following: + [diskStatus-ActiveGood.png] — indicates disk is active or good. + [diskStatus-Standby.png] — indicates disk is on standby. + [diskStatus-Offline.png] — indicates disk is offline. Mnt Indicates whether ( [checkMark.png] ) or not the disk is mounted. Type The type of disk. Value can be: + System — disk for system use only + MapR-FS — disk available for MapR-FS Device The disk partition(s). File System Specifies how the disk is being used. Value of MapR-FS indicates disk has been added to MapR-FS for file system to use. Allocated The amount of space allocated, in gigabytes. Used The percentage of disk space used. Model# The disk model number. Firmware Version The disk firmware version. Storage Pool The ID of the storage pool associated with the disk. Select the checkbox beside a disk of type MapR-FS to: + [5]Add Disk(s) to MapR-FS + [6]Remove Disk(s) from MapR-FS Note: Disks of type System cannot be selected. Retrieving the List of Disks Using the CLI or REST API The basic command to list the disks on a node is: maprcli disk list -host For complete reference information, see [7]disk list.
