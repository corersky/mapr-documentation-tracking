Viewing Node Details You can view node details using MCS and the CLI. Viewing Node Details Using MCS 1. Log in to MCS and click Nodes. 2. Click the hostname of the node. The information page for the node displays. + [1]Summary tab (default view) + [2]Metrics tab You can: + [3]Change the topology of the node + [4]Remove the node Viewing Node Details Using the CLI or REST API The basic command to retrieve information on a node is: maprcli node metrics -nodes -columns For complete reference information, see [5]node metrics.
