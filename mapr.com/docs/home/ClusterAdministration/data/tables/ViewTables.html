Viewing the List of Tables You can retrieve and view the list of tables using MCS and the CLI. Viewing the Tables in a Volume Using the MapR Control System 1. Log in to MCS and click Data > Tables to view all the volumes that you have access to. For each volume, the pane displays the following: Column Name Column Description Name The name of the volume. Type The type. Value can be: + [volPath.png] — Volume + [dirPath.png] — Directory + [tablePath.png] — Table Owner The name of the owner. Last Modified The last modification date and timestamp. 2. Click on the name of the volume (to browse to the path to the table) or enter the name of the volume in the text field. The tables in the selected volume display. If necessary, click the name of the directory to browse further or [homeView.png] to return to All volumes view. Viewing a Table by Table Path Using the MapR Control System 1. Log in to MCS and click Data > Tables. 2. Enter the path to the table and click GO. The tables information page for the specified table displays. Listing the Tables in a Directory From the Command-line Method for Binary Tables Only (HBase Shell) After starting the HBase shell, run the list command. Include the directory path in the command if you want to list tables that are not in your home directory. Type help to see a list of commands and their syntax. Method for JSON Tables Only (mapr dbshell) After starting the shell, run the list command. Include the directory path in the command if you want to list tables that are not in your home directory. See [1]MapR Database Shell (JSON Tables). Retrieving a Table by Table Path Using the CLI or the REST API To retrieve table details for a table by specifying the table path from the CLI, run the following command: maprcli table info -path For information on this command, see [2]table info.
