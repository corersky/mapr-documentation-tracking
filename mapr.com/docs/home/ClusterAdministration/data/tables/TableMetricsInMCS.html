Table Metrics in the MapR Control System A subset of the following primary table and secondary index metrics are available as charts and lists in the MapR Control System. For information on how to: * View these metrics in the MCS, see [1]Monitoring Tables. Metric Chart/Column Name Description Throughput - bytes read Table Bytes Read Per Node The number of bytes read from the primary table per node for all RPC types. Throughput - bytes read Table and Index Bytes Read The number of bytes read across a primary table and its secondary indexes for all RPC types. Throughput - bytes written Table Bytes Written Per Node The number of bytes written to the primary table per node for all RPC types. Throughput - bytes written Table and Index Bytes Written The number of bytes written across a primary table and its secondary indexes for all RPC types. Throughput - rowCount read Table Rows Read Per Node The number of rows read from the primary table per node for all RPC types. Throughput - rowCount read Table and Index Rows Read The number of rows read across a primary table and its secondary indexes for all RPC types. Throughput - rowCount written All Tables Written Rows Throughput Number of rows written by RPC operation type. Throughput - rowCount written Table Rows Written Per Node The number of rows written to the primary table per node for all RPC types. Throughput - rowCount written Table and Index Rows Written The number of rows written across a primary table and its secondary indexes for all RPC types. Throughput - rowCount returned Table Rows Responded Per Node The number of rows returned from the primary table per node for all RPC types. Throughput - rowCount returned Table and Index Rows Responded The number of rows returned across a primary table and its secondary indexes for all RPC types. Throughput - rowCount returned Scan Throughput Compares the scan throughput for rows read versus rows returned. Throughput - rpcCount/second All RPCs per Node The number of completed operations of all types across all tables and indexes for a node. Throughput - rpcCount/second All Tables RPC Byte Throughput The number of bytes processed by RPC operation type. Throughput - rpcCount/second All Tables Read Rows Throughput Number of rows processed by RPC operation type. Throughput - rpcCount/second All Tables Returned Rows Throughput Number of rows returned by RPC operation type. Throughput - rpcCount/second All Tables Throughput Number of RPC operations by type for all tables in the cluster. Throughput - rpcCount/second Put and Append Operation Throughput Number of put and append RPC operations for the table, including its indexes. Throughput - rpcCount/second Table Check and Put Ops Per Node The number of check and put operations completed for a primary table and for a node. Throughput - rpcCount/second Table Update and Get Ops Per Node The number of update and get operations completed for a primary table and for a node. Throughput - rpcCount/second Table Get and Index Scans The number of get and index scans completed for a primary table and for a node. Throughput - rpcCount/second Table Get Ops Per Node The number of get operations completed for a primary table and for a node. Throughput - rpcCount/second Table Get Throughput Per Node The number of get operations completed per second for a table, excluding its secondary indexes, per node. Throughput - rpcCount/second Table Increment Ops Per Node The number of increment operations completed for a primary table and for a node. Throughput - rpcCount/second Table Put Ops Per Node The number of put operations completed for a primary table and for a node. Throughput - rpcCount/second Table Scan Ops Per Node The number of scan operations completed for a primary table and for a node. Throughput - rpcCount/second Table Write Throughput Per Node The number of put and append operations per second completed for a table, excluding its secondary indexes, per node. Throughput - rpcCount/second Table and Index Put Ops Per Node The number of put operations completed per secondary index of the primary table. Throughput - rpcCount/second Table and Index Put Ops The number of put operations completed for the primary table and all its secondary indexes. Throughput - rpcCount/second Table and Index Scan Ops The number of scan operations completed for the primary table and all its secondary indexes. Throughput - rpcCount/second Table and Index Scan Ops Per node The number of scan operations completed per secondary index of the primary table. Latency Table and Index Scan Latency The 99th percentile latency of all scan operations across the primary table and its secondary indexes. A bad ratio between rows read and responded with high scan latency may indicate a poorly configured index. Latency Table and Index Scan Latency Per node The 99th percentile latency of scan operations completed across the primary table and secondary index per node. Large scans may hit the disks and result in poor performance, or a degrading disk may spike the latency. Latency Table Append Latency Per Node The 99th percentile latency of append operations on the primary table per node. Latency Table Increment Latency Per Node The 99th percentile latency of increment operations on the primary table per node. Latency Table Put Latency Per Node The 99th percentile latency of put operations on the primary table per node. Latency Table Get Latency Per Node The 99th percentile latency of get operations on the primary table per node. Latency Table Scan Latency Per Node The 99th percentile latency of scan operations on the primary table per node. Latency Primary Table Put & Append Latency Percentiles^* The pure write operation latency by percentile for the primary table and its secondary indexes. Latency Table Get Latency Percentiles The get operation latency by percentile for the primary table and its secondary indexes. Latency Table Write Throughput Latency Percentiles The 99th percentile latency of put operations on the primary table per node. Latency Table Get Throughput Latency Percentiles The 99th percentile latency of get operations on the primary table per node. Latency Table Check and Put Latency Per Node The 99th percentile latency of check and put operations on the primary table per node. Latency Table Update and Get Latency Per Node The 99th percentile latency of update and get operations on the primary table per node. Replication - rows/bytes sent All Tables Replication Activity Number of bytes of replication data sent vs not yet sent. CDC - rows/bytes pending All Tables CDC Pending Bytes The number of bytes of CDC data per node not yet sent. CDC - rows/bytes pending All Tables CDC Pending Rows The number of rows of CDC data not yet sent. Streams Throughput, RPCs All Streams Producer Ops The number of Streams producer RPCs. Streams Throughput, RPCs All Streams Consumer Ops The number of Streams consumer RPCs. Streams Throughput, messages All Streams Producer Messages The number of Streams messages produced. Streams Throughput, messages All Streams Consumer Messages The number of Streams messages read by consumers. Value Cache Lookups Table Value Cache All Lookups All operations for a primary table and for a node that performed a cache lookup. Value Cache Lookups Table Value Cache Lookups The number of get operations for a primary table and for a node that performed a cache lookup. Value Cache Hits Table and Index Value Cache All Lookups All operations across the primary table and its secondary indexes that performed a cache lookup. Value Cache Hits Table and Index Value Cache Lookups The number of get operations across the primary table and its secondary indexes that performed a cache lookup. Value Cache Hits Table Value Cache All Hits All operations for a primary table and for a node that resulted in a cache hit. Value Cache Hits Table Value Cache Hits The number of get operations for a primary table and for a node that resulted in a cache hit. Value Cache Hits Table and Index Value Cache All Hits All operations across the primary table and its secondary indexes that resulted in a cache hit. Value Cache Hits Table and Index Value Cache Hits The number of get operations across the primary table and its secondary indexes that resulted in a cache hit. Memindex Usage All Tables Free Index Memory The number of free MB in an in-memory cache for tables. ^* Percentiles are estimated by linearly interpolating between fixed buckets sizes.
