Renaming a Volume You can rename a volume using the MapR Control System, CLI, and REST API. If you rename a volume, you must unmount and re-mount the volume to allow applications and/or users to continue accessing the volume. Renaming a Volume Using the MapR Control System 1. Log in to MCS and go the [1]volume information page. 2. Click Rename Volume from the Select Action drop-down menu to display the Rename Volume window. 3. Enter the new name for the volume in the New Volume Name field and click Save Changes for the changes to take effect. Note: For tiering-enabled volumes, volume name cannot exceed ninety-eight characters. Renaming a Volume Using the CLI and REST API * [2]CLI * [3]REST The basic command to rename a volume is: maprcli volume rename -name -newname Send a request of type POST. For example: curl -k -X POST 'https://:8443/rest/volume/rename?name=&newname=< newName>' --user mapr:mapr See [4]volume rename for more information.
