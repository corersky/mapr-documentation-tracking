Viewing the list of Snapshots shortdesc You can view the snapshots on the cluster using MCS or the CLI. Viewing All the Snapshots Using the MapR Control System The Snapshots tab under Data > Volumes displays all the snapshots on the cluster. For each snapshot, you can see see the following: CAPTION: Column Name Column Description Snapshot Name The name of the snapshot. Volume The volume that the snapshot is associated with. Created On The date the snapshot was created. Expires On The date when the snapshot will expire. You can select one or more snapshots to: * [1]Preserve * [2]Remove Viewing the Snapshots Associated with a Volume Using the MapR Control System Log in to MCS and go to the Snapshots tab in the [3]volume information page. The list of snapshots associated with the volume displays in this tab. For each snapshot, the pane displays the following: CAPTION: Column Name Column Description Snapshot Name The name of the snapshot. Volume The volume that the snapshot is associated with. Created On The date the snapshot was created. Expires On The date when the snapshot will expire. You can [4]create a snapshot of the volume or select one or more snapshots to: * [5]Preserve * [6]Remove Viewing Snapshots Using the CLI or REST API The basic command to retrieve a list of snapshots is: maprcli volume snapshot list For complete reference information, see [7]volume snapshot list.
