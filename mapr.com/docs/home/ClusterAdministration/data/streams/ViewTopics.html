Viewing the List of Topics in a Stream You can view the list of topics in a stream using MCS and the CLI. Viewing the List of Topics in a Stream Using the MapR Control System Log in to MCS and go to the Topics tab in the [1]stream information page. The All topics pane displays the list of topics in the stream and for each topic, the pane displays the following: Column Name Column Description Topic Name The name of the topic. Maximum Lag The consumer lag time (in milliseconds). Partitions The number of partitions in the topic. Consumers The number of consumers for the topic. Physical Size The physical size (in MB) of the topic. You can view a topic from the list of topics by entering the topic name in the search field. You can also: * [2]Add a topic to the stream. * [3]Remove one or more topics. * [4]Modify the number of partitions for a topic. Viewing the List of Topics in a Stream Using the CLI or REST API The command to view the list of topics in a stream is: stream topic list -path For complete reference information, see [5]stream topic list.
