Removing Topics in a Stream You can delete one or more topics from the stream using MCS and the CLI. Consumers do not have to stop consuming from a topic before the topic is deleted. Removing Topics in a Stream Using the MapR Control System 1. Log in to MCS and the go to the Topics tab in the [1]stream information page. 2. Select the topics to remove in the All topics pane and click Remove Topic(s). The Remove Topic(s) confirmation window displays. 3. Verify the list of topics and click Remove Topic. The topic and the messages are immediately deleted. Removing Topics in a Stream Using the CLI or REST API The command to delete a topic is: maprcli stream topic delete -path -topic For complete reference information, see [2]stream topic delete.
