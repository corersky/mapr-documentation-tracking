Enabling and Disabling Auditing of Cluster Administration You can enable or disable auditing of cluster-management operations on a MapR cluster using MCS and the CLI. See [1]Auditing Cluster Activity for the complete list of cluster management commands that are audited. Enabling and Disabling Auditing of Cluster Administration Using MCS 1. Log in to MapR Control System and click Admin > Cluster Settings > Auditing. 2. Move the Enabled slider to Yes to enable and No to disable cluster auditing. 3. Click Save Changes for the changes to take effect. Enabling and Disabling Auditing of Cluster Administration Using the CLI or REST API To enable or disable auditing of cluster-management operations on a MapR cluster, run the maprcli audit cluster command. maprcli audit cluster -enabled For complete reference information, see [2]audit cluster.
