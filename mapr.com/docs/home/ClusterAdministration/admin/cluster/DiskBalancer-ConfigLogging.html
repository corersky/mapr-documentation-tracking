Configuring Logging The disk balancer logs information about all the bins and the storage pools in those bins in the /opt/mapr/logs/cldbdiskbalancer.log file every 10 minutes by default. The frequency for logging information can be configured by setting a custom value for the configurable parameter, dbal.loadtracker.info.log.frequency. Alternatively, the information can be automatically logged every time disk balancer runs by setting the value for the parameter cldb.balancer.logging to 1. Run the maprcli config save command to set one of the following parameters: CAPTION: Parameter Description dbal.loadtracker.info.log.frequency Sets custom logging schedule for disk balancer. The default value is 10 minutes. cldb.balancer.logging Enables or disables logging every time disk balancer runs. Value can be: * 0 — disable logging * 1 — enable logging
