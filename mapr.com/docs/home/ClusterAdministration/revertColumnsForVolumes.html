Reverting to Default List of Columns 1. Log in to MCS and click: + Data > Volumes to revert to the default list of columns in the Volumes pane. + Nodes to revert to the default list of columns in the Nodes pane. 2. Select Customize Columns from the [gearDropDownMenu.png] dropdown menu. 3. Select the Reset to default columns check box and click Save Changes. The pane displays the default list of columns.
