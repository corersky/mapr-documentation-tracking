Customizing the List of Metric Charts and Columns in the MapR Control System You can customize the list of table metric charts and columns that you see in the MapR Control System. Adding and Removing Charts from the Charts View 1. Log in to MCS and go to one of the following pages: + Metrics tab of the [1]node details page. + Metrics tab in the Data > Tables page. + Metrics tab of the [2]table information page. + Metrics tab of the [3]secondary index details page. 2. Click [whitePencil.png] to display the Customize Active Charts window. In the Customize Active Charts window, the: + Available list displays the charts available, but currently not displayed. + Selected list displays the charts currently displayed in the page. 3. Select the charts from the: a. Selected list of charts and click [lesserThan.png] to remove selected charts. b. Available list of charts and click [greaterThan.png] to move selection to Selected charts to display on page. You can select up to six charts to display at a time on the page. For the list of charts that can be viewed, see [4]Table Metrics in the MapR Control System. 4. (Optional) Click [upArrow.png] and/or down [downArrow.png] arrows to sort the order of charts. 5. Click Save to view the selected charts. Adding and Removing Columns from the List View 1. Log in to MCS and do one of the following: + Go to the Metrics tab of the [5]node details page and select Activity by Tables from the dropdown menu to view the metrics for all table-related activities on the node. + Go to the Metrics tab of the [6]table information page and select: o Activity by Nodes (default selection) from the dropdown menu to view the metrics for table activity across nodes. o Activity by Indexes from the dropdown menu to view metrics for all index-related activity on the table. + Go to the Metrics tab of the [7]secondary index details page and select Activity by Nodes to view metrics for index-related activity across nodes. 2. Click [listView.png] to switch to a list view. 3. Click [gearDropDownMenu.png] to display the Customize Columns window. In the Customize Columns window, the: + Available list displays the columns available, but currently not displayed. + Selected list displays the columns currently displayed in the page. 4. Select the columns from the: a. Selected list of columns and click [lesserThan.png] to remove selected columns from the view. b. Available list of columns and click [greaterThan.png] to move selection to Selected columns (for display). For the list of metrics that can be viewed in the columns, see [8]Table Metrics in the MapR Control System 5. (Optional) Click [upArrow.png] and/or down [downArrow.png] arrows to sort the order of columns. 6. Click Save to view the selected columns.
