Prerequisites to Running the MapR Container for Developers To run the MapR Container for Developers, you must first install the MapR client and Docker software. The instructions in this topic are specific to Mac OS X. The container is supported on only this platform. 1. Install a MapR client on your laptop: a. Download the MapR 6.1 client from [1]https://package.mapr.com/releases/v6.1.0/mac/ b. Extract the client software to the /opt directory by running the following commands: sudo mv ~/Downloads/mapr-client-6.1.0.20180926230239.GA-1.x86_64.tar.gz /opt cd /opt sudo tar xvfz mapr-client-6.1.0.20180926230239.GA-1.x86_64.tar.gz 2. Install Docker on your laptop: a. Download the software from [2]https://www.docker.com/docker-mac b. Install the software c. Verify that Docker is running with at least 8 GB of memory by clicking on Docker->Preferences->Advanced: [DockerPreferences.png] 3. Modify the memory settings, if needed, and restart Docker: [DockerMemory.png] References 1. https://package.mapr.com/releases/v6.1.0/mac/ 2. https://www.docker.com/docker-mac
