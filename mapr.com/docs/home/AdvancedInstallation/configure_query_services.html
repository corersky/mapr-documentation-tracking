Configure the OJAI Distributed Query Service Use these steps to install and configure the [1]OJAI Distributed Query Service: 1. Using the MEP 4.0 or later repository, install Drill to run under Warden on all data nodes. See [2]Install Drill to Run Under Warden. 2. Configure the query service by running the following configure.sh script command on the Drill nodes: /opt/mapr/server/configure.sh -R -QS 3. Register the query service: Note: In the following command is the name of the cluster as specified in /opt/mapr/conf/mapr-clusters.conf. maprcli cluster queryservice setconfig -enabled true -clusterid -dr illbits -storageplugin dfs -znode /drill References 1. file://localhost/root/docsync/tmp/mapr.com/docs/home/MapR-DB/ArchitectureOJAIQueryService.html#concept_pc2_4ws_p1b 2. file://localhost/root/docsync/tmp/mapr.com/docs/home/AdvancedInstallation/install_drill_warden.html#InstallingApacheDrillonMa_27689700-d3e67
