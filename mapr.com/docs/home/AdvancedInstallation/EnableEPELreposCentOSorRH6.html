Enable the EPEL repository on CentOS 6.x, RedHat 6.x, or Oracle Linux 6.4 or higher This section describes how to download and install the EPEL repository. 1. On a computer that is connected to the internet, download the EPEL repository: wget http://dl.fedoraproject.org/pub/epel/epel-release-latest-6.noarch.rpm 2. Install the EPEL repository: rpm -Uvh epel-release-6*.rpm
