Supported Clients To manually perform tiering-related operations on a volume, you can use the following: * The [1]maprcli command * The [2]REST API To manually perform tiering-related operations on a file, you can use the following: * The [3]FUSE-based POSIX client client * The [4]loopbacknfs POSIX client client * The [5]NFS client * The [6]hadoop command * The [7]maprcli command * The [8]REST API You must use clients from MapR v6.1 for accessing tiered volumes and performing tiering operations. You cannot use mixed mode clients to access and run tiering jobs on tiered volumes.
