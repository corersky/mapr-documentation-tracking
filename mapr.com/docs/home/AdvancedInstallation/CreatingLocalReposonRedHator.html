Creating a local repository on RedHat, CentOS, or Oracle Linux This section describes how to create and use a local repository. 1. Log in as root on the node or use sudo. 2. Create the following directory if it does not exist: /var/www/html/yum/base 3. On a computer that is connected to the internet, download the following files, substituting the appropriate number and : (See [1]MapR Repositories and Package Archives for the correct paths for all past releases.) https://package.mapr.com/releases/v6.x.x/redhat/mapr-GA.rpm.tgz https://package.mapr.com/releases/MEP/MEP-/redhat/mapr-mep--.rpm.tgz 4. Copy the files to /var/www/html/yum/base on the node, and extract them there. tar -xvzf mapr-vGA.rpm.tgz tar -xvzf mapr-mep--.rpm.tgz 5.
