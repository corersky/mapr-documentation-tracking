MapR Repositories and Packages Describes the repositories for MapR software and Hadoop Ecosystem tools. Repositories for MapR Core Software MapR hosts rpm and deb repositories for installing the MapR core software using Linux package-management tools. For every release of the core MapR software, a repository is created for each supported platform. Platform-specific installation repositories are hosted at: [1]https://package.mapr.com/releases/v6.x.x/. Repositories for MapR Ecosystem Packs A MapR Ecosystem Pack (MEP) provides a set of ecosystem components that work together. MapR hosts rpm and deb repositories for installing the ecosystem components, such as Flume, HBase, Hive, Myriad, Oozie, Pig, and Sqoop. These platform-specific repositories are hosted at the following location: * [2]https://package.mapr.com/releases/MEP/MEP-/ For more information about the MapR Ecosystem Packs (MEP), see [3]MapR Ecosystem Packs. GitHub Repositories for Source Code MapR releases the source code for Hadoop ecosystem components to GitHub, including all patches that MapR has applied to the components. MapR repositories under GitHub include Cascading, Flume, HBase, HCatalog, Hive, Myriad, Oozie, Pig, Sqoop, and Whirr. Source code for all releases since March 2013 are available at [4]https://github.com/mapr. Maven Repositories for Application Developers MapR hosts a Maven repository where application developers can download dependencies on MapR software or Hadoop ecosystem components. Maven artifacts for all releases since March 2013 are available at [5]Maven Artifacts for MapR.
