Applying a Patch Using a MapR Installer Stanza Applying a patch using a MapR Installer Stanza leverages the automation provided by the patch-install capability of the web-based MapR Installer. To apply a patch using a MapR Installer Stanza, you specify a file name and directory in the environment.patch_location parameter in the Stanza (YAML) file. Then you issue the install command to run the Stanza. Note: Applying a patch using the MapR Installer is an offline update (not a rolling update). For information about the install command, see [1]Installing or Upgrading MapR Core Using a MapR Installer Stanza. For information about the Stanza parameters, see [2]Working with MapR Installer Stanza Files.
