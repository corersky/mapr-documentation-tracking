Applying a Patch You can apply a patch by using the MapR Installer, by using the command line (a manual process), or by using a MapR Installer Stanza. To download a MapR patch: MapR Release Download Location For Patch Information 5.2.1 or later 1. Navigate to the MapR secure FTP server at [1]https://sftp.mapr.com. 2. Log in using maprpatches for your Login ID. Leave the Password field blank. 3. Click Login. Contact your support representative. 5.2.0 or earlier [2]http://package.mapr.com/patches/releases/ See [3]Patch Release Notes. If you need more information or if you face any problems with patch installation, contact [4]https://mapr.com/support. References 1. https://sftp.mapr.com/ 2. http://package.mapr.com/patches/releases/ 3. https://mapr.com/docs/home/home/PatchRN.html 4. https://mapr.com/support
