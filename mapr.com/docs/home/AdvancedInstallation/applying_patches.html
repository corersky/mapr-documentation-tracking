Applying a Patch You can apply a patch by using the MapR Installer, by using the command line (a manual process), or by using a MapR Installer Stanza. To download a MapR patch: 1. Navigate to the MapR secure FTP server at [1]https://sftp.mapr.com. 2. Log in using maprpatches for your Login ID. Leave the Password field blank. 3. Click Login. For patch information, contact your support representative or visit [2]https://mapr.com/support. References 1. https://sftp.mapr.com/ 2. https://mapr.com/support
