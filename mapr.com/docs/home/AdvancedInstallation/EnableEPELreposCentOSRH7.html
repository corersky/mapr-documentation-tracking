Enable the EPEL repository on CentOS 7.x, RedHat 7.x, or Oracle Linux 7.0/7.1 This section describes how to download and install the EPEL repository. 1. Download the EPEL repository: wget http://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm 2. Install the EPEL repository: rpm -Uvh epel-release-7*.rpm
